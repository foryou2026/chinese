## REF-1：飞书（Feishu / Lark）

- **借这一点**：侧边栏导航的分组折叠交互；顶栏搜索与用户菜单的简洁布局；亮色模式的纯白净底；表格的紧凑高效排列
- **不借**：过多的蓝色品牌色露出

## REF-2：Notion

- **借这一点**：极简的黑白按钮对比（亮色黑按钮/暗色白按钮）；干净的 1px 边框分隔；暗色模式的 #191919 深灰底；克制的排版与间距
- **不借**：Block 编辑器的内容排版模式

## REF-3：Linear（linear.app）

- **借这一点**：键盘可达 100% 主流程；表格 `tabular-nums` 数字对齐；高信息密度的列表视图；暗色模式的精致感
- **不借**：过于工程化的冷感

## REF-4：Vercel Dashboard

- **借这一点**：纯黑 Primary 按钮的视觉冲击力；极简的表单设计；清晰的状态标签（Badge）系统
- **不借**：开发者专属的技术气质
