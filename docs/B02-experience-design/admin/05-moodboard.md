## 关键词（6 个，带可验证含义）

| 形容词 | 可验证含义 | 反例 |
|-------|----------|------|
| 干净利落 | 浅灰底色 + 克制 Frost 毛玻璃卡片（blur ≤ 28px, saturate ≤ 1.5），无 mesh blob | app 的高饱和极光毛玻璃 |
| 黑白极致 | 亮色 Primary 按钮纯黑 #18181B；暗色 Primary 按钮纯白 #FAFAFA | 品牌色渐变按钮 |
| 信息密实 | 默认表格行高 40px、正文 14px、卡片 padding 16px | 大留白极简 SaaS |
| 快速响应 | 所有交互 ≤ 150ms 反馈，无装饰性过渡动画 | 弹跳动效 / 庆祝动画 |
| 层级分明 | 背景 → 卡片 → 弹层，通过背景色深浅和边框区分，不依赖阴影 | 大阴影多层次 |
| 深邃双模 | dark mode 使用纯黑 #000000 底（OLED 友好）+ zinc 冷灰阶卡片，毛玻璃层级 | 蓝灰底或简单反色 |
