## 整体调子

**像一个温暖但不卖萌的游戏教练。** 鼓励但不浮夸，直接但不冷漠。成功时给出正向数值奖励（+10 XP），失败时告诉原因并引导重试。用游戏化语言（关卡、连胜、成就）包装学习过程。

## 对照表

| 场景 | 反例 | 推荐（app） | 原因 |
|------|------|-----------|------|
| 加载中 | "正在飞速为你加载中..." | "加载中" | 不卖萌，尊重等待 |
| 提交成功 | "太棒了！你真厉害！" | "已保存" | 不过度感叹 |
| 练习正确 | "Wow! Amazing!" | "正确 +10 XP" | 给数值奖励而非空洞赞美 |
| 练习错误 | "哎呀，你答错啦~" | "不对。正确答案：X" | 直说 + 给正确答案 |
| 完成关卡 | "恭喜你通关了！！！" | "关卡完成！+50 XP" | 数值激励 + 克制 |
| 连胜提醒 | "快来学习吧不然就断了哦！" | "连胜 7 天。今天继续？" | 陈述事实 + 简短邀请 |
| 必填校验 | "您似乎漏掉了什么呢？" | "请填写邮箱" | 指出具体字段 |
| 服务器错误 | "Oops! 出了点小问题" | "服务异常，请稍后重试" | 给下一步 |
| 网络断开 | "网络好像不太好哦~" | "网络已断开。恢复后自动重连" | 告知状态 + 自动行为 |
| 空数据 | "这里空空如也~" | "暂无记录。开始第一课" | 给行动路径 |
| 删除确认 | "你确定要删除吗？" | "删除后无法恢复。确认删除？" | 说明后果 |
| 解锁新关卡 | "新关卡已解锁~" | "新关卡已解锁" | 不加 emoji 波浪 |
| 升级 | "你升级啦！！" | "等级提升 → Lv.5" | 陈述事实 |

## 语气边界

| 维度 | 规则 |
|------|------|
| 称呼 | "你" |
| emoji | 仅成功/庆祝态允许少量使用（如连胜火焰 🔥），其他场景禁止 |
| 感叹号 | 最多 1 个，禁止连续感叹号（!!!） |
| 游戏化术语 | 使用：XP、连胜、关卡、等级；不使用：生命值、死亡、失败（改为"未通过"） |

---

## 多语言语气原则

| 语言 | 注意事项 |
|------|---------|
| zh（中文） | 基准语气，友好直接 |
| en（英文） | 与中文对齐，避免过度 casual（如 "awesome!""oops!"） |
| vi（越南语） | 使用 "bạn"（你）而非 "quý khách"（贵客），保持亲近 |
| th（泰语） | 使用 ครับ/ค่ะ 礼貌词尾，但不过度谦卑 |
| id（印尼语） | 使用 "kamu" 而非 "Anda"（您），对齐年轻用户习惯 |
