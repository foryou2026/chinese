## REF-1：Duolingo

- **借这一点**：关卡路径式学习进度（蜿蜒地图）；XP / 连胜 / 宝石等游戏化货币系统；答对庆祝动效（星星飞散）；答错震动 + 红色闪烁；大号圆角按钮的触感设计；底部 Tab 导航的简洁分区
- **不借**：过度卡通化的吉祥物插画风格；过饱和的荧光色（我们走高级饱和度）

## REF-2：HelloChinese

- **借这一点**：中文学习场景的交互范式（拼音点读、笔画练习的 UI 模式）；课程单元的卡片式分组；学习进度的视觉化表达
- **不借**：略显陈旧的 UI 设计；扁平无层次的卡片风格

## REF-3：Apple visionOS Glass

- **借这一点**：玻璃面板的"近实远虚"层级感；`backdrop-filter` 的边缘高光（顶部 1px 暖白渐变）；分层玻璃的深度系统；导航栏的半透明悬浮感
- **不借**：3D 空间感 / 深度推拉动效；过大圆角

## REF-4：Memrise

- **借这一点**：太空/冒险主题的沉浸感；学习内容与游戏化的深度融合；丰富的正确答案反馈动效
- **不借**：过于写实的主题包装（太空飞船等）

## REF-5：Arc Browser

- **借这一点**：多彩渐变主题色系统（用户可自选主题色）；大胆的色彩运用不失优雅；年轻、有活力的品牌调性
- **不借**：桌面端专属的空间布局

## REF-6：Raycast

- **借这一点**：暗色模式下的毛玻璃质感；微交互的弹性缓动（spring easing）；极快的视觉反馈（<120ms）
- **不借**：纯工具导向的功能密度
