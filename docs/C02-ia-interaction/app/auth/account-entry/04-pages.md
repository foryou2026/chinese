| P-app-auth-003 | 找回密码 | 忘记密码页 | form | 公开 | M-auth-001 | R-auth-005, R-auth-011 | 无 | [本轮新增] |

## P-app-auth-001 登录页

### 用户目标
已注册用户通过邮箱密码或 Google 快捷方式登录。

### 进入条件与初始数据形态
- 未登录状态访问
- 无初始数据

### 信息架构（区块布局 ASCII）
```
+--------------------------------------------------+
|                   页面全屏                         |
|  +--------------------------------------------+  |
|  |              LOGO + 应用名                   |  |
|  |              欢迎回来                         |  |
|  |                                            |  |
|  |  [邮箱输入框]                                |  |
|  |  [密码输入框] [显示/隐藏]                     |  |
|  |  [登录按钮]                                  |  |
|  |  [忘记密码?]                                 |  |
|  |                                            |  |
|  |  ──── 或 ────                               |  |
|  |                                            |  |
|  |  [Google 登录按钮]                           |  |
|  |                                            |  |
|  |  还没有账号？注册                             |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+
```

| Block-ID | 名称 | 职责 | OP-ID | 隐藏条件 |
|----------|------|------|-------|---------|
| BLOCK-BRAND | 品牌区 | Logo+应用名+欢迎语 | 无 | 无 |
| BLOCK-LOGIN-FORM | 登录表单 | 邮箱密码输入 | OP-001~005 | 无 |
| BLOCK-DIVIDER | 分割线 | "或"分割 | 无 | 无 |
| BLOCK-SOCIAL | 第三方登录 | Google 按钮 | OP-006 | 无 |
| BLOCK-SWITCH | 切换链接 | 跳转注册 | OP-007 | 无 |

### 操作清单
| OP-ID | 名称 | 主/次 | 触发 | 期望反馈 | 角色 | 边界 |
|-------|------|-------|------|---------|------|------|
| OP-001 | 输入邮箱 | 主 | input | 失焦校验格式 | 公开 | 无 |
| OP-002 | 输入密码 | 主 | input | 无实时校验 | 公开 | 无 |
| OP-003 | 切换密码可见 | 次 | click | 图标切换 | 公开 | 无 |
| OP-004 | 点击登录 | 主 | click | loading→成功跳转/错误提示 | 公开 | 空值禁用 |
| OP-005 | 忘记密码 | 次 | click | 跳转 P-app-auth-003 | 公开 | 无 |
| OP-006 | Google 登录 | 主 | click | mock 弹窗→成功跳转 | 公开 | 无 |
| OP-007 | 去注册 | 次 | click | 跳转 P-app-auth-002 | 公开 | 无 |

### 状态四件套
| 状态 | 触发条件 | 应表达 | 主操作 |
|------|---------|-------|-------|
| 默认 | 页面加载 | 空表单 | OP-004/006 |
| 加载中 | 点击登录/Google | 按钮 loading+禁用 | — |
| 空数据 | 不适用 | — | — |
| 错误 | 认证失败 | Alert 错误 | 重试 |

### 场景验证
| SC-ID | GIVEN | WHEN | THEN | 涉及 OP/SM |
|-------|-------|------|------|-----------|
| SC-001 | 登录页 | 输入正确邮箱密码点击登录 | 跳转首页 | OP-004/SM-auth-001 |
| SC-002 | 登录页 | 密码错误 | 显示"邮箱或密码错误" | OP-004 |
| SC-003 | 失败 4 次 | 第 5 次错误 | 锁定 30 分钟 | OP-004/SM-auth-001 |
| SC-004 | 登录页 | 点击 Google 登录 | mock 弹窗→登录成功跳转 | OP-006/SM-auth-001 |
| SC-005 | 登录页 | 邮箱格式错误后失焦 | 内联错误 | OP-001 |
| SC-006 | 提交中 | 再次点击 | 不可重复提交 | OP-004 |
| SC-007 | 登录页 | 点击"还没有账号？注册" | 跳转注册页 | OP-007 |

### 表单
| 字段 | 必填 | 校验 | 错误位置 |
|------|------|------|---------|
| 邮箱 | 是 | 非空+格式 | 字段下方 |
| 密码 | 是 | 非空 | 字段下方 |

### 键盘/a11y
- Tab：邮箱→密码→登录→忘记密码→Google→注册
- Enter 提交

### 移动端
- 全屏居中卡片，宽度 100% - 32px
- 触控 ≥ 44px

### 信息架构
```
+--------------------------------------------------+
|  +--------------------------------------------+  |
|  |              LOGO + 应用名                   |  |
|  |              创建你的账号                     |  |
|  |                                            |  |
|  |  [邮箱输入框]                                |  |
|  |  [密码输入框] [显示/隐藏]                     |  |
|  |  密码强度指示器                               |  |
|  |  [确认密码输入框]                             |  |
|  |  [注册按钮]                                  |  |
|  |                                            |  |
|  |  ──── 或 ────                               |  |
|  |                                            |  |
|  |  [Google 注册按钮]                           |  |
|  |                                            |  |
|  |  已有账号？登录                               |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+
```

### 操作清单
| OP-ID | 名称 | 主/次 | 触发 | 期望反馈 |
|-------|------|-------|------|---------|
| OP-010 | 输入邮箱 | 主 | input | 失焦校验 |
| OP-011 | 输入密码 | 主 | input | 实时强度指示器 |

| OP-013 | 点击注册 | 主 | click | loading→自动登录跳转 |

### 场景验证
| SC-ID | GIVEN | WHEN | THEN | 涉及 OP/SM |
|-------|-------|------|------|-----------|

| SC-014 | 注册页 | 点击 Google 注册 | mock→自动创建+登录 | OP-014/SM-auth-001 |

## P-app-auth-005 修改/设置密码页

### 用户目标
已登录用户修改密码（邮箱用户）或设置密码（Google 用户）。

### 信息架构
```
+--------------------------------------------------+
|  ← 返回设置                                       |
|  +--------------------------------------------+  |
|  |  修改密码 / 设置密码                          |  |
|  |                                            |  |
|  |  [旧密码] ← 仅邮箱用户可见                    |  |
|  |  [新密码] + 强度指示器                        |  |
|  |  [确认新密码]                                 |  |
|  |  [确认修改按钮]                               |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+
```

### 场景验证
| SC-ID | GIVEN | WHEN | THEN |
|-------|-------|------|------|
| SC-030 | 邮箱用户 | 输入正确旧密码+合规新密码 | Toast 成功，返回设置页 |
| SC-031 | 邮箱用户 | 旧密码错误 | 显示旧密码错误 |
| SC-032 | 邮箱用户 | 新旧相同 | 显示不能与旧密码相同 |
| SC-033 | Google 用户 | 无旧密码，输入新密码 | Toast 密码已设置 |
| SC-034 | 任何用户 | 密码强度不足 | 内联错误 |
| SC-035 | 任何用户 | 确认不一致 | 内联错误 |

## P-app-auth-006 设置页

### 用户目标
查看账号信息，进入修改密码或退出登录。

### 信息架构
```
+--------------------------------------------------+
|  [BottomBar: 首页 | 学习 | 我的(当前)]            |
|  +--------------------------------------------+  |
|  |  头像 + 昵称 + 邮箱                          |  |
|  +--------------------------------------------+  |
|  |  账号安全                                    |  |
|  |    修改密码 / 设置密码                   [>]  |  |
|  +--------------------------------------------+  |
|  |  关于                                       |  |
|  |    版本信息                             [>]  |  |
|  +--------------------------------------------+  |
|  |  [退出登录]  ← 红色文字                      |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+
```

### 场景验证
| SC-ID | GIVEN | WHEN | THEN |
|-------|-------|------|------|
| SC-040 | 设置页 | 点击修改密码 | 跳转 P-app-auth-005 |
| SC-041 | 设置页 | 点击退出登录 | 确认弹框→退出→跳转登录页 |
| SC-042 | Refresh Token 过期 | 系统检测 | 弹窗→重新登录 |
