# 业务流程图

## FL-auth-001 账号入口：邮箱密码登录

涉及模块：M-auth-001 / 涉及状态机：SM-auth-001

```mermaid
flowchart LR
  A1[打开登录页] --> B1[输入邮箱+密码]
  B1 --> C1[点击登录]
  C1 --> D1{前端校验}
  D1 -->|格式错误| E1[内联错误]
  E1 --> B1
  D1 -->|通过| F1[提交认证]
  F1 --> G1{后端校验}
  G1 -->|成功| H1[跳转首页]
  G1 -->|密码错误| I1[显示错误+累计次数]
  I1 --> J1{≥5次?}
  J1 -->|是| K1[锁定30分钟]
  J1 -->|否| B1
  G1 -->|账号不存在| L1[显示错误]
  L1 --> B1
```

## FL-auth-003 账号入口：Google 快捷登录

```mermaid
flowchart LR
```

## FL-auth-006 账号入口：修改密码

涉及模块：M-auth-001

```mermaid
flowchart LR
  A6[我的→设置→修改密码] --> B6[进入修改密码页]
  B6 --> C6{Google用户?}
  C6 -->|是| D6[显示设置密码表单-无旧密码]
  C6 -->|否| E6[显示修改密码表单-含旧密码]
  D6 --> F6[输入新密码+确认]
  E6 --> G6[输入旧密码+新密码+确认]
  F6 --> H6{校验}
  G6 --> H6
  H6 -->|失败| I6[内联错误]
  I6 --> B6
  H6 -->|通过| J6[提交]
  J6 --> K6{后端}
  K6 -->|成功| L6[Toast成功→返回设置]
  K6 -->|旧密码错误| M6[显示错误]
```
