# 业务流程图

## FL-auth-001 账号入口：邮箱密码登录

涉及模块：M-auth-001 / 涉及状态机：SM-auth-001

```mermaid
flowchart LR
  A1[打开登录页] --> B1[输入邮箱+密码]
  B1 --> C1[点击登录]
  C1 --> D1{前端校验}
  D1 -->|格式错误| E1[内联错误]
  E1 --> B1
  D1 -->|通过| F1[提交认证]
  F1 --> G1{后端校验}
  G1 -->|成功| H1[跳转首页]
  G1 -->|密码错误| I1[显示错误+累计次数]
  I1 --> J1{≥5次?}
  J1 -->|是| K1[锁定30分钟]
  J1 -->|否| B1
  G1 -->|账号不存在| L1[显示错误]
  L1 --> B1
```

## FL-auth-002 账号入口：邮箱注册（含邮箱验证码）

涉及模块：M-auth-001 / 涉及状态机：SM-auth-001

```mermaid
flowchart LR
  A2[打开注册页] --> B2[步骤1-输入邮箱+密码+确认密码]
  B2 --> C2[点击下一步]
  C2 --> D2{前端校验}
  D2 -->|校验失败| E2[内联错误]
  E2 --> B2
  D2 -->|通过| F2{邮箱检查}
  F2 -->|邮箱已注册| G2[提示已存在+引导登录]
  F2 -->|邮箱可用| H2[步骤2-发送验证码+显示验证码输入框]
  H2 --> I2[输入6位验证码]
  I2 --> J2{验证码校验}
  J2 -->|正确| K2[注册成功→自动登录→跳转首页]
  J2 -->|错误| L2[显示验证码错误]
  L2 --> I2
  J2 -->|过期| M2[提示已过期，需重新发送]
```

## FL-auth-003 账号入口：Google 快捷登录

涉及模块：M-auth-001 / 涉及状态机：SM-auth-001

```mermaid
flowchart LR
  A3[点击Google登录] --> B3[弹出Google授权窗口]
  B3 --> C3{用户授权?}
  C3 -->|拒绝/关闭| D3[返回登录页]
  C3 -->|同意| E3{已有账号?}
  E3 -->|是| F3[直接登录→跳转首页]
  E3 -->|否| G3[自动创建账号→登录→跳转首页]
  B3 -.->|Google服务不可用| H3[显示降级提示]
```

## FL-auth-004 账号入口：忘记密码（发送验证码）

涉及模块：M-auth-001

```mermaid
flowchart LR
  A4[点击忘记密码] --> B4[进入忘记密码页-步骤1]
  B4 --> C4[输入邮箱]
  C4 --> D4{校验}
  D4 -->|格式错误| E4[内联错误]
  E4 --> C4

  F4 --> G4[进入步骤2-显示验证码输入框+60秒倒计时]
  G4 --> H4[输入6位验证码]
  H4 --> I4{验证码校验}
  I4 -->|正确| J4[跳转步骤3-重置密码页]
  I4 -->|错误| K4[显示验证码错误]
  K4 --> H4
  I4 -->|过期| L4[提示已过期，需重新发送]
```

## FL-auth-006 账号入口：修改密码

涉及模块：M-auth-001

```mermaid
flowchart LR
  A6[我的→设置→修改密码] --> B6[进入修改密码页]
  B6 --> C6{Google用户?}
  C6 -->|是| D6[显示设置密码表单-无旧密码]
  C6 -->|否| E6[显示修改密码表单-含旧密码]
  D6 --> F6[输入新密码+确认]
  E6 --> G6[输入旧密码+新密码+确认]
  F6 --> H6{校验}
  G6 --> H6
  H6 -->|失败| I6[内联错误]
  I6 --> B6
  H6 -->|通过| J6[提交]
  J6 --> K6{后端}
  K6 -->|成功| L6[Toast成功+其他设备已下线→返回设置]
  K6 -->|旧密码错误| M6[显示错误]
  K6 -->|新旧密码相同| N6[显示不能与旧密码相同]
```
