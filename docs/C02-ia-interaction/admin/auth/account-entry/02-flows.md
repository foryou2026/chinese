# 业务流程图

## FL-auth-001 账号入口：邮箱密码登录

涉及模块：M-auth-001 / 涉及状态机：SM-auth-001

```mermaid
flowchart LR
  A1[打开登录页] --> B1[输入邮箱+密码]
  B1 --> C1[点击登录]
  C1 --> D1{前端校验}
  D1 -->|格式错误| E1[内联错误提示]
  E1 --> B1
  D1 -->|通过| F1[提交认证请求]
  F1 --> G1{后端校验}
  G1 -->|凭证正确+role=admin| H1[跳转 Dashboard]
  G1 -->|密码错误| I1[显示错误+累计失败次数]
  I1 --> J1{失败≥5次?}
  J1 -->|是| K1[锁定30分钟提示]
  J1 -->|否| B1
  G1 -->|role≠admin| L1[显示无权限提示]
  G1 -->|账号不存在| M1[显示邮箱或密码错误]
  M1 --> B1
```

## FL-auth-002 账号入口：忘记密码（发送验证码）

涉及模块：M-auth-001

```mermaid
flowchart LR
  A2[点击忘记密码] --> B2[进入忘记密码页-步骤1]
  B2 --> C2[输入邮箱]
  C2 --> D2{前端格式校验}
  D2 -->|格式错误| E2[内联错误]
  E2 --> C2
  D2 -->|通过| F2[提交发送验证码]
  F2 --> G2[进入步骤2-显示验证码输入框+60秒倒计时]
  G2 --> H2[输入6位验证码]
  H2 --> I2{验证码校验}
  I2 -->|正确| J2[跳转步骤3-重置密码页]
  I2 -->|错误| K2[显示验证码错误]
  K2 --> H2
  I2 -->|过期| L2[提示已过期，需重新发送]
```

## FL-auth-003 账号入口：重置密码

涉及模块：M-auth-001 / 涉及状态机：SM-auth-001

```mermaid
flowchart LR
  A3[验证码校验通过] --> B3[进入重置密码页-步骤3]
  B3 --> C3[输入新密码+确认密码]
  C3 --> D3{前端校验}
  D3 -->|强度不足/不一致| E3[内联错误]
  E3 --> C3
  D3 -->|通过| F3[提交重置]
  F3 --> G3{后端处理}
  G3 -->|成功| H3[显示成功+其他设备已下线+3秒跳转登录页]
  G3 -->|失败| I3[显示错误提示]
```

## FL-auth-004 账号入口：修改密码

涉及模块：M-auth-001

```mermaid
flowchart LR
  A4[点击头像→修改密码] --> B4[打开修改密码弹框]
  B4 --> C4[输入旧密码+新密码+确认密码]
  C4 --> D4{前端校验}
  D4 -->|校验失败| E4[内联错误]
  E4 --> C4
  D4 -->|通过| F4[提交修改]
  F4 --> G4{后端校验}
  G4 -->|旧密码错误| H4[显示旧密码错误]
  H4 --> C4
  G4 -->|新旧相同| I4[显示新密码不能与旧密码相同]
  I4 --> C4
  G4 -->|成功| J4[Toast成功+其他设备已下线+关闭弹框]
```
