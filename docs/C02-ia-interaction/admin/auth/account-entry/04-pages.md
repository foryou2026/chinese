## P-admin-auth-001 登录页

### 用户目标
管理员输入邮箱和密码，完成身份验证并进入后台。

### 进入条件与初始数据形态
- 未登录状态直接访问或被路由守卫重定向
- 无初始数据

### 信息架构（区块布局 ASCII）
```
+--------------------------------------------------+
|                   页面全屏居中                      |
|  +--------------------------------------------+  |
|  |              LOGO + 系统名称                  |  |
|  |                                            |  |
|  |  [邮箱输入框]              BLOCK-LOGIN-FORM  |  |
|  |  [密码输入框] [显示/隐藏]                     |  |
|  |  [登录按钮]                                  |  |
|  |  [忘记密码链接]                               |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+
```

| Block-ID | 名称 | 职责 | OP-ID | 隐藏条件 |
|----------|------|------|-------|---------|
| BLOCK-LOGIN-HEADER | 品牌标识 | 展示 Logo 和系统名称 | 无 | 无 |
| BLOCK-LOGIN-FORM | 登录表单 | 收集邮箱密码、触发认证 | OP-001~004 | 无 |
| BLOCK-LOGIN-FOOTER | 辅助链接 | 提供忘记密码入口 | OP-005 | 无 |

| OP-005 | 点击忘记密码 | 次 | click | 跳转 P-admin-auth-002 | 公开 | 无 |

| 空数据 | 不适用 | — | — |
| 错误 | 认证失败 | 表单上方 Alert 错误（密码错误/无权限/锁定） | OP-004 重试 |

## P-admin-auth-003 重置密码页

### 用户目标
通过邮件链接设置新密码。

### 进入条件与初始数据形态
- 从邮件中点击重置链接进入，URL 携带 Token
- 页面加载时校验 Token 有效性

### 信息架构（区块布局 ASCII）
```
+--------------------------------------------------+
|  +--------------------------------------------+  |
|  |           设置新密码                          |  |
|  |                                            |  |
|  |  [新密码输入框] [显示/隐藏]                    |  |
|  |  密码强度指示器                               |  |
|  |  [确认密码输入框]                             |  |
|  |  [确认重置按钮]                               |  |
|  +--------------------------------------------+  |
|                                                  |
|  --- Token无效时 ---                              |
|  +--------------------------------------------+  |
|  |           链接已失效                          |  |
|  |  重置链接已过期或无效                          |  |
|  |  [返回登录] [重新获取]                        |  |
|  +--------------------------------------------+  |
+--------------------------------------------------+
```

| Block-ID | 名称 | 职责 | OP-ID | 隐藏条件 |
|----------|------|------|-------|---------|
| BLOCK-RESET-FORM | 重置表单 | 收集新密码 | OP-020~024 | Token 无效时隐藏 |
| BLOCK-RESET-EXPIRED | 链接失效 | 提示过期+引导 | OP-025~026 | Token 有效时隐藏 |
| BLOCK-RESET-SUCCESS | 重置成功 | 成功提示+倒计时跳转 | 无 | 初始隐藏 |

### 操作清单
| OP-ID | 名称 | 主/次 | 触发 | 期望反馈 | 角色 | 边界 |
|-------|------|-------|------|---------|------|------|
| OP-020 | 输入新密码 | 主 | input | 实时强度指示器更新 | 公开 | 无 |
| OP-021 | 切换密码可见 | 次 | click | 眼睛图标切换 | 公开 | 无 |
| OP-022 | 输入确认密码 | 主 | input | 实时一致性校验 | 公开 | 无 |
| OP-023 | 确认重置 | 主 | click | loading→成功提示→3s 跳转登录 | 公开 | 强度不足/不一致禁用 |
| OP-025 | 返回登录 | 次 | click | 跳转 P-admin-auth-001 | 公开 | 无 |
| OP-026 | 重新获取 | 次 | click | 跳转 P-admin-auth-002 | 公开 | 无 |

### 状态四件套
| 状态 | 触发条件 | 应表达 | 主操作 |
|------|---------|-------|-------|
| 默认 | Token 有效 | 空密码表单 | OP-023 |
| 加载中 | 提交重置 | 按钮 loading | — |
| 空数据 | 不适用 | — | — |
| 错误 | Token 无效/过期 | 链接失效视图 | OP-025/026 |

### 场景验证
| SC-ID | GIVEN | WHEN | THEN | 涉及 OP/SM |
|-------|-------|------|------|-----------|
| SC-020 | Token 有效 | 输入合规新密码+确认一致，点击确认 | 显示"密码重置成功"，3 秒后跳转登录页 | OP-023/SM-auth-001 |
| SC-021 | Token 过期 | 页面加载 | 显示"链接已失效"视图 | 无 |
| SC-022 | 新密码 < 8 字符 | 失焦 | 显示"密码至少 8 位，包含字母和数字" | OP-020 |
| SC-023 | 确认密码不一致 | 失焦 | 显示"两次输入的密码不一致" | OP-022 |
| SC-024 | 密码仅含数字 | 失焦 | 显示密码强度不足 | OP-020 |
| SC-025 | 正在提交 | 再次点击 | 按钮禁用 loading 态 | OP-023 |

### 表单
| 字段 | 必填 | 校验规则 | 错误位置 |
|------|------|---------|---------|
| 新密码 | 是 | ≥8字符，含字母+数字 | 字段下方内联 |
| 确认密码 | 是 | 与新密码一致 | 字段下方内联 |

### 键盘/a11y
- Tab：新密码→确认密码→确认重置
- Enter 提交

### 移动端
- 同登录页布局策略

- 展示 Dashboard 骨架 + 顶栏用户信息

|               |                                   |
+--------------------------------------------------+

+------------------+
| 修改密码          |
| ───────────────  |
| 退出登录          |
+------------------+

修改密码弹框：
+--------------------------------------------+
|  修改密码                              [×]  |
|  [旧密码输入框]                              |
|  [新密码输入框]  密码强度指示器               |
|  [确认新密码输入框]                          |
|  [取消]  [确认修改]                          |

+--------------------------------------------+
