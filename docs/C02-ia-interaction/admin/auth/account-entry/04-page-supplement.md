## P-admin-auth-001 登录页

### 表单校验规则
| 字段 | 必填 | 校验规则 | 错误文案 |
|------|------|---------|---------|
| 邮箱 | 是 | 非空 + 邮箱格式 | 空："请输入邮箱"；格式错误："请输入有效的邮箱地址" |
| 密码 | 是 | 非空 | "请输入密码" |

### 登录失败分级提示
| 失败次数 | 错误文案 |
|---------|---------|
| 第 1-2 次 | 邮箱或密码错误 |
| 第 3-5 次 | 邮箱或密码错误（已失败 N/5 次） |
| 第 5 次后 | 登录失败次数过多，账号已锁定 30 分钟 |
| 角色不符 | 您没有管理后台访问权限 |

### 键盘/a11y
- Tab 顺序：邮箱→密码→登录→忘记密码
- Enter 提交；所有 input 有 label for

## P-admin-auth-002 忘记密码页

### 表单校验规则
| 字段 | 必填 | 校验规则 | 错误文案 |
|------|------|---------|---------|
| 邮箱 | 是 | 非空 + 邮箱格式 | 同登录页 |
| 验证码 | 是 | 6 位数字 | 错误："验证码错误"；过期："验证码已过期，请重新发送" |

### 键盘/a11y
- Tab：邮箱→发送→验证码(6格)→验证→重新发送
- Enter 提交

## P-admin-auth-003 重置密码页

### 表单校验规则
| 字段 | 必填 | 校验规则 | 错误文案 |
|------|------|---------|---------|
| 新密码 | 是 | ≥8 字符，含字母+数字 | "密码至少 8 位，包含字母和数字" |
| 确认密码 | 是 | 与新密码一致 | "两次输入的密码不一致" |

### 键盘/a11y
- Tab：新密码→确认密码→确认重置
- Enter 提交

---

## P-admin-auth-004 后台壳

### 修改密码弹框校验规则
| 字段 | 必填 | 校验规则 | 错误文案 |
|------|------|---------|---------|
| 旧密码 | 是 | 非空 | "请输入旧密码"；后端："旧密码错误" |
| 新密码 | 是 | ≥8 字符，含字母+数字，≠旧密码 | 强度不足同上；相同："新密码不能与旧密码相同" |
| 确认新密码 | 是 | 与新密码一致 | "两次输入的密码不一致" |

### 键盘/a11y
- Escape 关闭弹框/菜单；Tab 在弹框内循环；弹框打开 body 锁滚

### 角色差异
无（仅 ROLE-ADMIN）
