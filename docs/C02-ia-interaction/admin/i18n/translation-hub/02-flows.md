# 业务流程图

## FL-i18n-001 语言启用流程

```mermaid
flowchart TD
    A[进入语言管理页] --> B[浏览全球语言列表]
    B --> C{选择语言}
    C --> D[切换启用开关]
    D --> E{是否为受保护语言?}
    E -->|中文/英文| F[提示：受保护语言不可停用]
    E -->|其他语言| G[保存状态到 i18n_languages]
    G --> H[app 端语言选择器同步更新]
    H --> I[该语言下所有翻译条目自动创建为 pending 状态]
```

## FL-i18n-002 文案翻译流程

```mermaid
flowchart TD
    A[进入文案翻译页] --> B[系统自动同步代码 key]
    B --> C[展示文案列表：中文原文 + 目标语言翻译]
    C --> D{操作选择}
    D -->|查看| E[切换目标语言 / 按域筛选 / 搜索]
    D -->|手动编辑| F[编辑翻译文案]
    F --> G[状态保持 translated]
    D -->|AI 翻译| H[选择翻译模型]
    H --> I{选择翻译范围}
    I -->|全部待翻译| J[创建翻译任务：全量]
    I -->|按语言| K[创建翻译任务：指定语言]
    I -->|按业务域| L[创建翻译任务：指定域]
    J & K & L --> M[按枢纽语言翻译链执行]
    M --> N[翻译完成，状态变为 translated]
    N --> O{是否发布?}
    O -->|是| P[同步到代码资源文件]
    P --> Q[app 端生效]
    O -->|否| R[保留在数据库，待发布]
```

## FL-i18n-006 配音任务流程

```mermaid
    B --> C{范围类型}

    L --> P[可重试失败部分]
    P --> H
```

## FL-i18n-007 枢纽语言翻译链流程

```mermaid
flowchart TD
    A[触发翻译请求] --> B{源语言是否为英文?}
    B -->|是| C[直接翻译 en → 目标语言]
    C --> D[保存翻译结果]
    B -->|否| E{该条目是否已有英文翻译?}
    E -->|已有| F[从英文翻译到目标语言]
    F --> D
    E -->|没有| G[先翻译源语言 → 英文]
    G --> H[保存英文翻译结果]
    H --> F
    D --> I[界面展示：源语言原文 ↔ 目标语言翻译对照]
```

> **规则要点**：
> 1. 英文（en）是唯一枢纽语言，任何语言间的翻译必须以英文为媒介
> 2. 翻译前先查询英文翻译是否存在，避免重复翻译
> 3. 界面上只展示源语言和目标语言的对照，英文中间过程对用户不可见
> 4. 如果源语言就是英文，直接翻译，无需中间步骤
