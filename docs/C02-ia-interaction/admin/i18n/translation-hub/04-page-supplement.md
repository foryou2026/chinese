## P-admin-i18n-001 翻译总览仪表盘

### 布局描述
顶部标题「翻译总览」。下方分两个区域：

**区域 1：语言概览卡片**
横向排列已启用语言卡片，每个卡片包含：
- 语言名称（中文）
- UI 翻译覆盖率（进度条 + 百分比）
- 内容翻译覆盖率（进度条 + 百分比）
- 待翻译数量 badge

**区域 2：快捷操作**
- 「全局翻译」按钮：翻译所有语言的所有待翻译项（UI + 内容）
- 「同步 UI 文案」按钮：从代码同步最新 key
- 各状态的数量统计：待翻译 / 已翻译 / 已审核 / 已过期

### 交互规则
- 点击语言卡片跳转该语言的翻译详情（UI 文案列表页，预选该语言）
- 「全局翻译」需确认弹框

## P-admin-i18n-002 语言管理页

### 布局描述
页面标题「语言管理」。下方为语言表格：

| 列 | 宽度 | 内容 |
|----|------|------|
| 排序手柄 | 40px | 拖拽图标（仅已启用的语言可拖拽） |
| 语言名称 | 200px | 中文名称（如"英语"、"越南语"） |
| 原生名称 | 200px | 该语言的本地名称（如"English"、"Tiếng Việt"） |
| Locale 代码 | 100px | `en`、`vi`、`th` 等 |
| 状态 | 100px | 启用/停用 Switch |
| 翻译覆盖率 | 150px | 进度条（仅已启用语言显示） |

### 预置语言列表（约 50 种主流语言）
按语系分组展示：

**东亚**：中文（zh）、日语（ja）、韩语（ko）
**东南亚**：越南语（vi）、泰语（th）、印尼语（id）、马来语（ms）、菲律宾语（fil）、缅甸语（my）、高棉语（km）、老挝语（lo）
**南亚**：印地语（hi）、孟加拉语（bn）、乌尔都语（ur）、泰米尔语（ta）、泰卢固语（te）、马拉地语（mr）
**西欧**：英语（en）、法语（fr）、德语（de）、西班牙语（es）、葡萄牙语（pt）、意大利语（it）、荷兰语（nl）
**北欧**：瑞典语（sv）、丹麦语（da）、挪威语（no）、芬兰语（fi）
**东欧**：俄语（ru）、波兰语（pl）、捷克语（cs）、罗马尼亚语（ro）、乌克兰语（uk）、匈牙利语（hu）
**中东/非洲**：阿拉伯语（ar）、土耳其语（tr）、希伯来语（he）、波斯语（fa）、斯瓦希里语（sw）
**美洲**：西班牙语-拉美（es-419）、葡萄牙语-巴西（pt-BR）

### 交互规则
- 中文行的启用开关显示为禁用态（灰色，tooltip：「系统默认语言，不可停用」）
- 启用新语言时弹出确认框：「启用{语言名}后，系统将为所有 UI 文案和已配置的数据库内容创建待翻译条目。确认启用？」
- 停用语言时弹出确认框：「停用{语言名}后，该语言的翻译内容将保留但不再在 app 端展示。确认停用？」
- 已启用语言置顶显示，可拖拽排序；未启用语言按语系分组在下方
- 排序变更即时保存
- 搜索框：支持按中文名、原生名、locale 代码搜索

## P-admin-i18n-003 UI 文案翻译列表页

### 布局描述

**顶部操作栏**
- 目标语言选择器（下拉菜单，默认「英语」，选项来自已启用语言，排除中文）
- 业务域筛选器（下拉菜单：全部 / common / auth / nav / course / ...，动态从 key 的一级前缀提取）
- 状态筛选器（全部 / 待翻译 / 已翻译 / 已审核 / 已过期）
- 搜索框（按 key 路径或中文原文搜索）
- 操作按钮组：「同步文案」「翻译筛选项」「翻译全部待翻译」「发布」

**主体表格**

| 列 | 宽度 | 内容 |
|----|------|------|
| Key 路径 | 200px | `auth.login`、`common.save` 等 |
| 中文原文 | flex | 源文案中文内容 |
| 目标翻译 | flex | 当前选中语言的翻译内容（可内联编辑） |
| 状态 | 100px | 状态标签（颜色区分） |
| 操作 | 80px | 编辑 / 翻译单条 |

**底部**
- 分页器（每页 50 条）
- 统计：共 N 条，已翻译 N 条，待翻译 N 条

### 交互规则
- 切换目标语言时表格右侧列刷新
- 内联编辑：点击翻译内容单元格即可编辑，失焦自动保存，状态变为 reviewed
- 「同步文案」：调用 API 从代码同步最新 key，新增 key 显示为 pending，已删除 key 标记为废弃（灰色斜体）
- 「翻译筛选项」：仅翻译当前筛选条件下的待翻译+已过期条目
- 「翻译全部待翻译」：翻译所有语言的所有待翻译+已过期条目
- 「发布」：将数据库中的翻译内容同步回代码资源文件，弹出确认框显示变更概览

### 分组展示
按业务域（key 的一级前缀）分组，每组可折叠：
```
▼ common（12 条）
  common.save    保存    Save    ✅ 已审核
  common.cancel  取消    Cancel  ✅ 已审核
  ...

▼ auth（8 条）
  auth.login     登录    Sign In  🟡 已翻译
  auth.register  注册    —       🔴 待翻译
  ...
```

## P-admin-i18n-005 表字段列表页

### 布局描述

**顶部面包屑**
数据库内容翻译 > {表名}

**表级配置概要区**
展示当前表的配置：翻译状态、默认源语言、总字段数、可翻译字段数

**主体表格**

| 列 | 宽度 | 内容 |
|----|------|------|
| 字段名 | 200px | title、description 等 |
| 字段类型 | 100px | text / varchar / jsonb 等 |
| 需要翻译 | 100px | 开关（默认继承表级，手动设置后覆盖） |
| 源语言 | 100px | 下拉选择（默认继承表级，手动设置后覆盖） |
| 需要配音 | 100px | 开关（可选模块，默认继承表级） |
| 翻译覆盖率 | 150px | 进度条（仅已启用翻译的字段显示） |
| 记录数 | 80px | 该表的总记录数 |
| 操作 | 80px | 进入翻译详情 |

### 交互规则
- 「需要翻译」开关变更后即时保存

- 仅 text/varchar/jsonb 类型字段可标记为需要翻译（数值/时间/布尔等类型灰色不可操作）
- 「源语言」下拉选项来自已启用语言列表
- 点击字段名或「进入翻译详情」跳转 P-admin-i18n-006

### 配置优先级说明
页面底部展示配置优先级规则卡片：
> 字段级配置 > 表级配置。若字段未手动设置，则继承表级默认值。手动设置后的字段显示 ⚙ 标记。

**操作栏**

- 状态筛选器
- 搜索框（按原文内容搜索）
- 「翻译全部待翻译」「翻译本页」

**主体表格**

| 列 | 宽度 | 内容 |
|----|------|------|
| 记录 ID | 100px | UUID 前 8 位 |
| 原文内容 | flex | 源语言的字段值（只读） |
| 翻译内容 | flex | 目标语言的翻译（可内联编辑） |
| 状态 | 100px | 状态标签 |
| 操作 | 80px | 编辑 / 翻译单条 |

**底部**
- 分页器（每页 50 条）

- 状态为 outdated 的行高亮显示（浅橙色背景）

## 通用键盘/a11y 规则
- Tab 按可视顺序遍历所有可交互元素
- 表格行 Enter 进入详情
- Switch 开关 Space 切换
- 弹框 Esc 关闭
- 触控目标 ≥ 44px
