## P-admin-i18n-001 翻译总览仪表盘

### 布局描述
页面分两个区域：

**区域 1：语言概览卡片**
横向排列已启用语言卡片，每个卡片包含：
- 语言名称（中文）
- UI 翻译覆盖率（进度条 + 百分比）
- 内容翻译覆盖率（进度条 + 百分比）
- 待翻译数量 badge

**区域 2：快捷操作**
- 「全局翻译」按钮：翻译所有语言的所有待翻译项（UI + 内容），需先选择翻译模型
- 「同步 UI 文案」按钮：从代码同步最新 key
- 各状态的数量统计：待翻译 / 已翻译

### 交互规则
- 点击语言卡片跳转该语言的翻译详情（文案翻译列表页，预选该语言）
- 「全局翻译」需确认弹框，弹框中需选择翻译模型

## P-admin-i18n-002 语言管理页

### 布局描述

| 列 | 宽度 | 内容 |
|----|------|------|
| 排序手柄 | 40px | 拖拽图标（仅已启用的语言可拖拽） |

### 预置语言列表（约 50 种主流语言）
按语系分组展示：

**东亚**：中文（zh）、日语（ja）、韩语（ko）
**东南亚**：越南语（vi）、泰语（th）、印尼语（id）、马来语（ms）、菲律宾语（fil）、缅甸语（my）、高棉语（km）、老挝语（lo）
**南亚**：印地语（hi）、孟加拉语（bn）、乌尔都语（ur）、泰米尔语（ta）、泰卢固语（te）、马拉地语（mr）
**西欧**：英语（en）、法语（fr）、德语（de）、西班牙语（es）、葡萄牙语（pt）、意大利语（it）、荷兰语（nl）
**北欧**：瑞典语（sv）、丹麦语（da）、挪威语（no）、芬兰语（fi）
**东欧**：俄语（ru）、波兰语（pl）、捷克语（cs）、罗马尼亚语（ro）、乌克兰语（uk）、匈牙利语（hu）
**中东/非洲**：阿拉伯语（ar）、土耳其语（tr）、希伯来语（he）、波斯语（fa）、斯瓦希里语（sw）
**美洲**：西班牙语-拉美（es-419）、葡萄牙语-巴西（pt-BR）

### 交互规则
- 中文行的启用开关显示为禁用态（灰色，tooltip：「系统默认语言，不可停用」）
- 英文行的启用开关显示为禁用态（灰色，tooltip：「枢纽语言，不可停用」），且种子数据中英文默认已启用
- 启用新语言时弹出确认框：「启用{语言名}后，系统将为所有 UI 文案和已配置的数据库内容创建待翻译条目。确认启用？」
- 停用语言时弹出确认框：「停用{语言名}后，该语言的翻译内容将保留但不再在 app 端展示。确认停用？」
- 已启用语言置顶显示，可拖拽排序；未启用语言按语系分组在下方
- 排序变更即时保存
- 搜索框：支持按中文名、原生名、locale 代码搜索

## P-admin-i18n-003 文案翻译列表页

### 布局描述

**顶部操作栏**
- 目标语言选择器（下拉菜单，默认「英语」，选项来自已启用语言，排除中文）
- 业务域筛选器（下拉菜单：全部 / common / auth / nav / course / ...，动态从 key 的一级前缀提取）
- 状态筛选器（全部 / 待翻译 / 已翻译）
- 搜索框（按 key 路径或中文原文搜索）
- 翻译模型选择器（下拉菜单，来自已配置的供应商模型列表）
- 操作按钮组：「同步文案」「翻译筛选项」「翻译全部待翻译」「发布」

**主体表格**

| 列 | 宽度 | 内容 |
|----|------|------|
| Key 路径 | 200px | `auth.login`、`common.save` 等 |
| 中文原文 | flex | 源文案中文内容 |
| 目标翻译 | flex | 当前选中语言的翻译内容（可内联编辑） |
| 状态 | 100px | 状态标签（待翻译🔴 / 已翻译🟢） |
| 操作 | 80px | 编辑 / 翻译单条 |

**底部**
- 分页器（每页 50 条）
- 统计：共 N 条，已翻译 N 条，待翻译 N 条

### 业务域提取机制
业务域筛选器的域从 key 的一级前缀自动提取（如 `common.save` → common, `auth.login` → auth），系统启动时扫描所有 i18n key 并自动构建域列表。域拆分用于方便管理员按功能模块筛选和批量翻译。

### 交互规则
- 切换目标语言时表格右侧列刷新
- 内联编辑：点击翻译内容单元格即可编辑，失焦自动保存，状态保持 translated
- 「同步文案」：调用 API 从代码同步最新 key，新增 key 显示为 pending，已删除 key 标记为废弃（灰色斜体）
- 「翻译筛选项」：使用选中的翻译模型，仅翻译当前筛选条件下的待翻译条目
- 「翻译全部待翻译」：使用选中的翻译模型，翻译所有语言的所有待翻译条目
- 「发布」：将数据库中的翻译内容同步回代码资源文件，弹出确认框显示变更概览
- 翻译模型选择器：来自已配置的供应商模型列表，默认选择标记为默认的模型

### 分组展示
按业务域（key 的一级前缀）分组，每组可折叠：
```
▼ common（12 条）
  common.save    保存    Save    🟢 已翻译
  common.cancel  取消    Cancel  🟢 已翻译
  ...

▼ auth（8 条）
  auth.login     登录    Sign In  🟢 已翻译
  auth.register  注册    —       🔴 待翻译
  ...
```

### 范围字段显示规则
翻译任务的「范围」列按以下规则生成可读文案：

| scope 类型 | scope_filter 内容 | 显示文案 | 示例 |
|-----------|------------------|---------|------|
| UI 文案 · 全局 | `{type:"ui", scope:"all"}` | `文案 · 全部` | 文案 · 全部 |
| UI 文案 · 按域 | `{type:"ui", scope:"namespace", namespace:"auth"}` | `文案 · {namespace}域` | 文案 · auth域 |
| UI 文案 · 选中项 | `{type:"ui", scope:"selected", count:5}` | `文案 · 选中{N}条` | 文案 · 选中5条 |
| 内容 · 全局 | `{type:"content", scope:"all"}` | `内容 · 全部` | 内容 · 全部 |
| 内容 · 按表 | `{type:"content", scope:"table", table_name:"courses"}` | `{表说明} · 全部字段` | 课程表 · 全部字段 |
| 内容 · 按字段 | `{type:"content", scope:"field", table_name:"courses", field_name:"title"}` | `{表说明} · {字段名}` | 课程表 · title |

### 交互规则
- 执行中的任务进度条自动刷新（WebSocket 或轮询 5 秒间隔）
- 点击任务行展开详情，展示失败条目列表
- 「取消」需确认弹框
- 「重试」仅重试失败的条目
