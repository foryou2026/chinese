# 系统兜底页

## 翻译 API 不可用

| 项 | 值 |
|----|-----|
| 触发条件 | AI 翻译接口返回 5xx 或超时 |
| 展现形式 | Toast 提示 + 翻译任务状态标记为失败 |
| 文案 | 翻译服务暂时不可用，请稍后重试 |
| 操作 | 翻译任务列表可重试 |

## TTS API 不可用

| 项 | 值 |
|----|-----|
| 触发条件 | TTS 接口返回错误 |
| 展现形式 | Toast 提示 + 配音任务标记为失败 |
| 文案 | 配音服务暂时不可用，请检查接口配置 |
| 操作 | 可重新生成或手动上传 |

## 代码同步失败

| 项 | 值 |
|----|-----|
| 触发条件 | 同步 UI 文案时后端读取代码资源文件失败 |
| 展现形式 | 全页错误提示 |
| 文案 | 文案同步失败，请联系开发人员检查资源文件 |
| 操作 | 重试按钮 |

## 通用网络错误

| 项 | 值 |
|----|-----|
| 触发条件 | 任何接口 4xx/5xx |
| 展现形式 | Toast 提示 |
| 文案 | 操作失败，请稍后重试 |
| 操作 | 保留当前页面状态 |
