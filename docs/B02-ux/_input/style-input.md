# 体验风格输入（B02-X01）

> 阶段：B02-X 体验定调 · 用户输入  
> 确认 by PM · 2026-04-28

---

## 1. 我们做的是什么

**知语 · Zhiyu** = 给东南亚（越南 / 泰国 / 印尼）成年学习者用的**汉语学习应用**。

- 目标用户：18–35 岁，东南亚母语 + 英语半流利，第一次系统学中文；少部分华人后裔回学母语。
- 使用场景：地铁上 5 分钟刷一段 / 晚上 30 分钟完整学一课 / 周末攒时间打游戏巩固。
- 核心承诺：**学得起 + 学得下去 + 觉得在玩**。

## 2. 用三句话定调

1. **大气、克制、专业**：参考 Stitch / Figma 的产品质感，不是花花绿绿的"儿童 App"。
2. **像在用一个高级社交产品在学习**：参考小红书 / Reddit，红色当唯一品牌色，毛玻璃当默认材质；让人愿意每天打开。
3. **游戏化但不幼稚**：进度、徽章、连胜、积分都有，但克制——不弹两层确认弹窗、不放彩带烟花、不堆 emoji。

## 3. 想成为谁（参照对象）

- **小红书**：红色品牌、毛玻璃、内容/卡片质感；
- **Stitch（Google 新设计工具）**：现代毛玻璃 + 节制的留白；
- **Notion**：内容区简洁、Markdown 感、键盘党友好。

## 4. 一定不要成为谁（反例）

- ❌ **多邻国（Duolingo）**：吉祥物当主视觉、五彩斑斓、卡通、儿童感。我们要"成年人"。
- ❌ **多数中文学习 App（HelloChinese / Hanly / SuperChinese）**：偏教辅、配色花、组件土。
- ❌ **传统 ToB 后台**：四四方方、灰底白卡、文字密集。我们的后台也得有"质感"。

## 5. 5 个可验证的形容词（任何设计决策必须能命中其中至少 3 个）

1. **现代** modern
2. **克制** restrained
3. **质感** tactile（毛玻璃 + 阴影边缘高光 + 微噪）
4. **趣味** playful（在适当处出现，不漫溢）
5. **沉浸** immersive（全屏宽度、远景光晕、暗模式真黑）

## 6. 语气与口吻

- **不耍宝**、不卖萌；
- 学习反馈直接、明确、温和：「再试一次」「做得不错」「这点很多人会错」；
- 错误提示**不羞辱**用户："邮箱或密码错误"而非"账号有问题"；
- 多语言：5 语全量；vi / th / id 文案不能直译，由 native 校对。

## 7. 互动哲学

1. **不打断**：能就地内联展示就不弹 Modal；
2. **不打扰**：成功不弹 Toast 也行（按钮回到 idle 即可），错误才打扰；
3. **不模糊**：每个状态有明确反馈，不让用户猜；
4. **不让用户等**：>300ms 必有骨架；>1s 必有进度；>5s 必能取消。

## 8. PM 签字
任何设计变更必须先回看本文，命中至少 3 个 §5 形容词，且不触犯 §4 反例。
