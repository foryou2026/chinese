## 1. 小红书 Xiaohongshu

### 我们参考
- **红色作为唯一品牌色**，且是"鲜红 + 偏冷"（小红书红 `#FF2442`），不是中国红；
- **卡片 + 毛玻璃 + 微圆角**的内容呈现节奏；
- 内容区**没有侧边栏**、信息流主导、上下滚动；
- 头像 / 入口的圆角 + 头像下拉的轻盈感；
- 移动端贴底操作 + 安全区适配。

### 我们不参考
- 双列瀑布流（小红书的杀手锏，但我们是学习而非内容消费）；
- 短视频 / 直播 入口（与产品无关）；
- 评论区的强社交（本期不做社交）；
- 商业化插入卡（学习场景里禁止）。

### 命中形容词
现代 ★★★★★ ・ 克制 ★★★★ ・ 质感 ★★★★★

## 2. Stitch（Google Labs 设计工具）

### 我们参考
- **毛玻璃材质感**：半透明面板 + `backdrop-filter` 模糊 + 顶部 1px 高光内边；
- **远景光晕**：背景的极低饱和径向渐变，制造前后景差；
- **微噪纹理**：毛玻璃面板上的隐约噪点，避免"塑料光滑感"；
- **动效曲线**：spring 风格、不超过 320ms、不抢戏；
- 顶栏滚动时的毛玻璃透出效果。

### 我们不参考
- Google 自家的过度"Material 3"圆角与渐变（我们更克制）；
- AI 生成预览的卡片堆叠交互（我们不是设计工具）。

### 命中形容词
现代 ★★★★★ ・ 质感 ★★★★★ ・ 沉浸 ★★★★

## 3. Notion

### 我们参考
- **内容区简洁**：长阅读型内容（小说、课程详文）用 prose 限宽 `max-w-[72ch]`，外层容器仍占满；
- **键盘党友好**：全局快捷键（`?` 弹帮助、`g h` 回首页、`/` 聚焦搜索）；
- **不打断**：编辑时不弹保存对话，离开自动保留草稿；
- **无侧边栏**也能组织复杂信息（我们用顶部导航 + 内联面包屑）；
- **Markdown 触感**：文本编辑器里的 inline command。

### 我们不参考
- Notion 的"无穷嵌套页面"（学习内容是平铺的）；
- 那种"什么都能塞进去"的灵活度（我们要场景明确）；
- 偏冷的灰白配色（我们是红 + 毛玻璃，更暖）。

### 命中形容词
现代 ★★★★ ・ 克制 ★★★★★
