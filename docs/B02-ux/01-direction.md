---

## 0. 一句话定调

> **"给东南亚成年学习者的、像高级社交产品一样使用的汉语学习应用。"**

任何设计决策的 reviewer 在 30 秒内能用这句话作为标尺：
- "这个屏看起来像不像高级社交产品？" —— 是 → 通过；不是 → 打回。
- "这个反馈适不适合成年人？" —— 是 → 通过；不是 → 打回。

---

## 1. 5 个可验证的形容词

> 顺序无关，但每一项必须能落到某个具体设计动作上。

| # | 形容词 | 含义 | 体现 |
|---|------|------|------|
| 1 | **现代** modern | 看起来像 2024+ 的主流产品，不是 2015 教辅 | 圆角 / 间距 / 字号取主流 SaaS；图标用 lucide；插画线条风格 |
| 2 | **克制** restrained | 颜色不超过 3 个层级、操作不超过 1 个主按钮 / 区段 | 唯一品牌色 = 红；每个卡片只有 1 个 primary action |
| 3 | **质感** tactile | 表面有"摸得到"的层次 | 毛玻璃 + 顶部 1px 高光内边 + 微噪 + 远景红色径向光晕 |
| 4 | **趣味** playful | 在"完成 / 连胜 / 升级"时有适度奖励 | 微动效（200–320ms spring）、进度条平滑、连胜徽章 |
| 5 | **沉浸** immersive | 进入页面像进入一个空间，不是看一张表 | 全屏宽度、远景光晕、暗模式真黑（#0A0A0B）、内容区随视口呼吸 |

**评分规则**：任何 C 阶段产物必须能命中至少 3 个；命中数写在 C04/C05 的 review 标签里。

---

## 2. 反向定调（绝不要的 5 件事）

| # | 绝不要 | 为什么 |
|---|------|------|
| 1 | **吉祥物当主视觉** | 立刻显得儿童化，跟"成年人" 冲突 |
| 2 | **彩色彩虹按钮 / 渐变 + emoji 满屏** | 跟"克制"冲突；视觉噪声大 |
| 3 | **多层弹窗（弹窗里再弹窗）** | 反"不打断"；让用户找不到上下文 |
| 4 | **空白卡片实色背景** | 反"质感"；毛玻璃失效；显得平面 |
| 5 | **小白条灰背景表格的传统后台** | 反"现代 + 沉浸"；管理端也得是同款语言 |

---

## 3. 决策路径（遇到犹豫时）

```
新设计 →
  ├─ 命中 ≥3 个形容词？  否 → 打回
  ├─ 是否触犯反向定调？  是 → 打回
  ├─ 是否破坏黑白红 + 毛玻璃？是 → 打回
  └─ 是否需要 > 300ms 思考？是 → 加骨架 / 加进度反馈
```

---

## 4. 与品牌 / 产品定位的一致性

- 品牌承诺："学得起 + 学得下去 + 觉得在玩"。
- 体验定调直接服务该承诺：
  - "学得起" → 克制（不让用户被界面吓到）；
  - "学得下去" → 不打断（学习心流）+ 沉浸（情境感）；
  - "觉得在玩" → 趣味（微动效 / 进度 / 连胜）+ 现代（颜值是日活的一部分）。
