## 0. 摘要

- 给谁：admin 端内容运营管理员（U2）。
- 做什么：媒资文件列表管理（列表 + 命中统计 + 软删）；按内容指纹去重；有引用时禁止删除。
- 最大价值：集中管理音频 / 图片资源，避免重复上传，确保被引用媒资不被误删。
- 最大风险：内容指纹碰撞 / 去重判断错误导致不同媒资被合并或删除。
