## 0. 摘要

- 给谁：admin 端内容运营管理员（U2）。
- 做什么：按主题 / 阶段 / 章分层配置考试蓝图；试抽预览（不计 attempt）；配置后学员端立即生效。
- 最大价值：让运营精确控制每层级考试的题目来源比例，保证考试质量。
- 最大风险：蓝图配置错误导致学员端考试题目比例异常。
