## 1. 范围

### 1.1 在范围内
- **5 主题骨架**:共享预备(share) + 电商(ec) + 工厂(fc) + HSK(hsk) + 日常(dl);
- **4 级层级**:主题 → 阶段 → 章 → 节,每章默认 6 节、每节默认 12 KP、节末 6 题;
- **7 类 KP** + **12 种题型**;**纯静默学习**(选 / 连 / 排序 / 点选 / 键盘),无麦无手写;
- **节内学习**:KP 卡片流 + 随堂练 + 节末小测(可跳过);
- **SRS Leitner 5 盒**复习 + 错题本;
- **考试中心**:节末小测 / 章测 / 阶段考 / HSK 模考(HSK 主题独享);
- **管理端 9 大页**:课程目录总览 / 主题-阶段-章-节列表 / 节编辑 / KP 列表 / 题目列表 / 学员举报 / 媒资库 / 考试中心 / 全局搜索;
- **内容生产闭环**:线下生成 → 批量导入(CSV/JSON)→ 人工点检 → 发布 → CDN;
- **层级级联发布**:章发布默认级联节 + KP + 题;节发布默认级联 KP + 题;章/节下架只下架自身层级;
- **5 语**内容字段(`zh/en/vi/th/id`);UI 应用端 5 语切换、管理端固定中文;
- **多端登录 + 进度服务端为准**;弱网答题先写本地队列;
- **媒资统一资产表** + hash 去重 + 引用拦截;
- **学员举报闭环**(P-A-6 处理 + 历史成绩不追溯);
- **版本号管理**:KP / Question 编辑 `version+1`,旧版本归档;
- **管理员双角色**:`super` / `content_admin` / `readonly`,行级 `tracks_scope` 控制。

### 1.2 不在范围内(v1)
- ❌ 实时 AI 对话陪练 / AI 发音打分 / AI 作文批改 / 千人千面个性化路径;
- ❌ 直播课、真人 1v1;
- ❌ 论坛 / UGC 笔记(二期);
- ❌ 系统内 AI 生成工作台(2025-11 废弃);
- ❌ 自定义主题码(永远 5 个固定);
- ❌ 题库自由建题(只能绑定已存在 KP);
- ❌ 阶段考重考(已封板 H5);
- ❌ 学员举报自动改内容(只人工)。

## 4. 边界与不变量

- **5 语 locale 集合**:`zh/en/vi/th/id`(与 [`B02-permissions/03-authz-mechanism.md §4`](../../B02-permissions/03-authz-mechanism.md) / discover-china C01 §4 全局对齐);多语字段统一形如 `title_i18n: { zh, en, vi, th, id }`,前端按 `users.ui_lang` 取值,缺失回退 `zh`;
- **5 主题码**:写死 `share / ec / fc / hsk / dl`,不接受业务方扩展;
- **节 code 格式**:`{track}-{stage}-{chapter}-{lesson}`(示例 `ec-2-3-1` / `share-0-1-1`);UNIQUE,迁移友好;
- **KP code 格式**:`kp_{track}_{type_initial}_{seq5}`(示例 `kp_ec_w_00231`,type_initial = p/h/w/ph/g/s/d);UNIQUE;
- **题目 code 格式**:`q_{track}_{seq8}`(示例 `q_ec_00012345`);UNIQUE;
- **数量基线**(运营预期):
  - 主题 5;阶段 25;章 ~148;节 ~888;KP ~12 000;Question ~5 万;音频 ~2 万;
  - 单节 KP 12(可 8-14);单 KP 配题 4-6;
- **节末小测**:6 题固定;`is_quiz_required=false` 可跳过;通过线 60%;
- **章测**:15 题(70% 本章 + 30% 前章 SRS);通过线 70%;
- **阶段考**:50 题均衡抽;通过线 70% 解锁下阶段;**默认不可重考**(决策 H5);
- **HSK 模考**:按官方题型比例,**HSK 主题独享**;
- **SRS 间隔表**:box 1→1 天;2→3 天;3→7 天;4→14 天;5→30 天;答对升一盒、答错回 box 1;
- **试抽**:`P-A-8` Drawer 内预览,不写 `user_exam_attempts`,不算 attempt;
- **题目下架动态均分**:试卷满分恒 100,单题分 = 100 / 当前有效已发布题数;已存 attempt snapshot 不变;
- **发布层级级联**:章发布默认级联节 + KP + Q;节发布级联 KP + Q;**下架不级联**(章下架只切章本身可见性,节级状态保留);
- **`is_published`二态**:`true`=已发布、`false`=待发布(含未发布 + 下架);无第三态;
- **版本治理**:KP / Question 编辑 `version+1`,旧版本归档;`user_answers` 历史不追溯;
- **媒资去重**:按 `hash`;删除前必拦截引用;
- **管理员行级权限**:`admins.tracks_scope` 数组控制可见主题;列表 SQL 自动 `WHERE track_code = ANY($scope)`;
- **舍弃项常态化**:无生成工作台、无 AI 提示词库、无任务编排页面(2025-11 决策);
- **管理端 UI 固定中文**;只有内容字段支持 5 语录入。
