# `discover-china` · 主流程（业务叙述）

> 流程图与详细状态机详见 [../../../C03-ia/discover-china/](../../../C03-ia/discover-china/)。本文仅以业务语言描述用户场景。

## 1. 访客首次正向体验

1. 访客打开 app，进入"发现中国" Tab。
2. 访客看到 12 张类目卡片。
3. 访客点击 01-03 任一类目，直接进入该类目文章列表。
4. 访客点开一篇文章，看到逐句对照（拼音 / 中文 / 当前语言）。
5. 访客点单句播放按钮，听到普通话朗读。

## 2. 登录学员深度阅读

1. 学员登录后再进同一文章，自动跳到上次阅读的句子。
2. 学员点击"全文朗读"，从当前句开始按顺序逐句播放。
3. 阅读进度跨设备同步。

## 3. 运营管理 12 类目

1. 运营在 admin 端打开"发现中国"，看到 12 个类目统计卡片。
2. 每张卡显示"已发布 / 总数"。
3. 运营选择类目，进入文章列表。

## 4. 运营新增 / 编辑文章

1. 运营在文章列表点击"新建"或选择已有文章进入编辑。
2. 在文章编辑页填写基本信息，进入句子分页。
3. 在任意句子位置使用"在开头 / 结尾 / 当前后插入"添加新句子。
4. 切换 5 语 Tab 录入对应语种文本（含拼音 / 中文校对）。
5. 通过拖拽对句子排序，保存后顺序号自动重排。
6. 保存成功后系统提示；如发生并发覆盖则以"后写覆盖 + 提示"告知。

## 5. 文章发布

1. 运营在文章列表对一篇文章执行"发布"。
2. 系统在发布前校验 5 语齐备。
3. 通过校验后状态变为"已发布"；学员端立即可见。
