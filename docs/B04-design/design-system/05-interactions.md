> **阶段**：B04-S  
> **上游**：`04-status-components.md`、`B03-ux/06-experience-principles.md`、`grules/G2-视觉与交互风格/05-通用交互.md`

## 0. 摘要

- 防重复提交：`useAsyncAction`；按钮立即 loading + disabled。
- 搜索：自动触发 + 300ms 防抖。
- 列表刷新：创建 / 编辑 = 重请求；删除 / 状态切换 = 乐观更新 + 失败回滚。
- 表单 dirty 离开必须保护；
- 分页默认 20 / 可选 10|20|50|100；URL 同步。
- 全局快捷键 `?` / `g h` / `g a` / `/` / `ESC`。
- 错误：4xx 内联 / 5xx 弹 Toast / 401 跳登录 / 403 跳 /403 / 404 跳 /404。

---

## 1. 防重复提交

- **方案**：所有提交按钮在请求未返回前立即进入 `loading + disabled` 双重态。
- **实现**：项目级 `useAsyncAction(fn)`：
  - 返回 `{ run, isLoading }`；
  - 重复调用在第一次未完成时被忽略（不排队）；
  - 异常自动 Toast（错误码 → i18n 文案）；
- **按钮约束**：所有提交按钮必须接收 `loading` prop；禁止自实现"点一次后变灰"；
- **表单整体**：表单未通过校验时主按钮 `disabled`。

---

## 2. 搜索

| 项 | 默认 |
|----|------|
| 触发 | **自动**（输入即查询）|
| 防抖 | **300ms** |
| 最小字符 | 1（中文 1，拉丁 2）|
| 清空 | 输入框右侧 X 图标，点击立即清空并刷新 |
| 历史 | **不保存**（隐私）|
| 占位 | i18n `common.search.placeholder`，含示例如 "搜索课程 / 拼音 / 标签" |

- 无结果 → 表格"未找到匹配项"空状态 + "清空筛选"按钮；
- 高级筛选位于搜索框右侧 `sliders` 抽屉触发图标。

---

## 3. 列表刷新策略

| 操作 | 默认策略 |
|------|---------|
| 创建成功 | 重请求当前页（保证排序 / 计数一致）|
| 编辑成功 | 重请求当前页 |
| 删除成功 | **乐观移除该行 + 后台请求**；失败插回 + Toast |
| 状态切换（启用 / 禁用、上下架）| 乐观更新 + 失败回滚 |
| 筛选 / 排序 / 翻页 | 重请求 |
| 跨页选中 | **不支持**（避免误操作）；当前页全选用 checkbox |

> 实现：统一 TanStack Query `invalidateQueries(['x','list',filters])`；禁止直接 `setState` 篡改列表。

## 4. 表单离开保护

- **场景**：编辑型表单（含 Modal 内）；
- **触发**：表单 dirty 且未保存；
- **行为**：
  - 同站路由跳转：弹"未保存的更改将丢失，确认离开吗？"确认弹窗；
  - 关闭 / 刷新：触发原生 `beforeunload`；
  - Modal / Drawer 关闭按钮 / 点遮罩（如允许）：同确认弹窗；
- **草稿**：≥ 30s 表单自动存 `localStorage`，key = `<feature>:<form-name>:<userId>`；进入页时恢复 + Toast "已恢复上次的草稿" + "清空"按钮；提交成功立即清；
- **实现**：`useFormGuard(form)`，订阅 React Hook Form `formState.isDirty`。

## 5. 分页

| 项 | 值 |
|----|---|
| 默认每页 | **20** |
| 可选 | 10 / 20 / 50 / 100 |
| 位置 | 表格底部右侧 |
| 显示 | 上一页 / 页码（最多 5 数字 + 省略号）/ 下一页 / 总数 / 每页选择 |
| 跳页 | 桌面"跳至第 N 页"；移动端隐藏 |
| 移动端 | 仅"上一页 / 当前 N / 下一页 / 共 X 条" |
| URL 同步 | `?page=2&pageSize=20` |

---

## 6. 键盘与无障碍

- **全局快捷键**：
  - `?`：弹"快捷键帮助" Modal；
  - `g h`：返回首页；
  - `g a`（管理端）：跳 `/admin`；
  - `/`：聚焦顶部搜索框；
  - `ESC`：关闭最上层弹窗 / Drawer / Toast；
- 任意可交互元素必须可 Tab 聚焦；focus ring 使用 `--ring`（红色半透明，不被毛玻璃吞没）；
- Modal 焦点陷阱由 Radix 提供；关闭后焦点回到触发按钮。

---

## 7. 复制 / 下载 / 上传

- **复制**：`navigator.clipboard.writeText`，成功后右上 Toast "已复制"；
- **下载**：隐藏 `<a>` + `URL.createObjectURL`；大文件后台生成 → 完成后 Toast 内提供下载按钮；
- **上传**：
  - Dropzone + 点击两种；拖入时容器 `--brand` 虚线 + 浅红底；
  - 文件类型 / 大小限制以 Toast 失败提示；
  - 上传中显示进度条（`--brand`），可取消。

---

## 8. 错误反馈（与 B03 §不模糊 / §给反馈 一致）

- **可恢复 4xx**：**内联**在表单 / 字段，**不弹 Toast**；
- **不可恢复 5xx / 网络**：右上 **Toast**，文案走 i18n（错误码 → `errors.50001` 等）；
- **未授权 401**：自动跳 `/auth/login?redirect=...`；
- **无权限 403**：跳 `/403` 毛玻璃页（含返回首页按钮）；
- **资源不存在 404**：跳 `/404` 毛玻璃页（含搜索建议）；
- 三件套文案规则见 [B03 §三 错误提示三件套](../../B03-ux/04-voice-tone.md)。

---

## 9. 动效

- **页面切换**：内容区淡入 120ms（`opacity 0→1`），无水平位移；
- **菜单 hover**：颜色变化 `--motion-fast`；
- **按钮 hover/active**：背景与阴影渐变 `--motion-base`；
- **弹窗 / Drawer 进出**：`--motion-slow`；
- **卡片 hover**：微微"上抬 + 阴影加深"（Apple Music 风）；
- 全部尊重 `prefers-reduced-motion: reduce`。

---

## 10. 学习场景的额外公约（汉语学习专属，源自 [B03 §4](../../B03-ux/06-experience-principles.md)）

1. 学错不羞辱（先承认难 → 给提示 → 给答案）；
2. 学对不夸张（朴素肯定，不放彩带）；
3. **打断学习心流是大罪**：学习页禁止任何与学习无关的弹窗 / 推送 / 营销；通知积压到学习结束后；
4. 题目过渡 ≤ 200ms；连续 5 题正确触发"连胜"微反馈；
5. 拼音 / 笔画展示：失误 ≥ 2 次自动提示，不让用户卡死；
6. 可暂停：学习活动（含游戏）能暂停且保留进度。
