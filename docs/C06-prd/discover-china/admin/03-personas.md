# 03 · 画像与场景 · discover-china / **admin**

## 3.1 主画像 · `super`

| 维度 | 设定 |
|------|------|
| 角色 | 项目核心 / 内容主理人;1-2 人 |
| 工作量 | 每周新增 5-10 篇文章;批量 TTS 重传 1-2 次 |
| 工作设备 | 桌面浏览器 |
| 主要任务 | 文章撰写 + 5 语校对 + TTS 上传 + 发布 |

## 3.2 `readonly`

- 可看类目 / 文章 / 句子;不可改;主要给项目外审计 / 顾问。

## 3.3 角色 × 模块矩阵

| 模块 | readonly | super |
|------|:--:|:--:|
| 分类 P-001 | 看 | 改名(5 语) |
| 文章列表 P-002 | 看 | 改 / 发布 / 撤回 |
| 文章编辑 P-003 | 看 | 改 / 上传 TTS |
| 搜索运维 P-004 | 看 | 触发重建 |

> 本 feature 无 `content_admin`;`super` 全权。
