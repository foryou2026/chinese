## 8. 关键场景（**重要**）

> 用 GIVEN/WHEN/THEN 写至少 5 个。AI 会用这些做交互验证。

```
SC-1: GIVEN 未登录用户进入课程详情
       WHEN 点击"加入课程"
       THEN 弹登录提示，登录成功后回到当前页继续

SC-2: GIVEN 已购用户进入
       WHEN 点击"开始学习"
       THEN 跳转到学习页（page-id=P-021）进入第一节

SC-3: GIVEN 列表数据为空
       WHEN 进入页面
       THEN 显示空态："暂无课程，去看看推荐" + 按钮跳转推荐页

SC-4: GIVEN 加载失败
       WHEN 进入页面
       THEN 显示错误态 + 重试按钮，非全屏

SC-5: GIVEN 移动端
       WHEN 进入页面
       THEN 主 CTA 固定在底部
```
