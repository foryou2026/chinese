## AI 必须遵守

1. **只读**：R 用户输入 + R-questions-round*-resolved + `A00-01` + `A00-03` + `A00-04` + 本模板。
   > **严禁读**：docs/B01-architecture/ 与任何开发阶段（数据 / 接口 / 校验）产出（C 与 D 隔离；R 阶段也不依赖 B01 架构）。
2. **不许做**：写技术选型、画详细业务流程图、定状态机、画 UI 草图、定字段、写接口。
   > **详细流程图与状态机是 C02 信息架构阶段的职责，R 阶段只叙述业务场景，不出 mermaid。**
3. 仍存疑的写到 §99，不要在正文猜。
4. 每条需求都有唯一 R-ID（格式 `R-<seq3>`）。
