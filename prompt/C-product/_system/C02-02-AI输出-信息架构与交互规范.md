# C02-02 AI 输出：信息架构与交互规范

> **阶段**：C02 架构与交互
> **归属**：**按系统独立（_system）**
> **谁产出**：AI（信息架构师 + 交互设计师）
> **落盘**：`docs/<system-id>/C02-ia-interaction/<module-id>/<feature-id>/`

---

## AI 必须遵守

1. **只读**：C02-01 输入 + C01 共享基线 + B02 体验规范 + 同系统已有 C02 + 本模板
2. **不许做**：URL / 路由 / API / 表字段 / HTML / 视觉设计
3. **不跨系统引用**：禁止引用其他系统的 C02 产出
4. 存疑写 §99
5. 流程图、状态机一律 mermaid
6. ID 规范：页面 `P-<system>-<module>-<seq3>`、模块 `M-<module>-<seq3>`、状态机 `SM-<module>-<seq2>`、操作 `OP-<page-id>-<seq>`
7. 每轮产出一个系统的一个功能，§10 列融合点和冲突点，标注 `[本轮新增]` / `[本轮变更]`

---
