## 六、编码阶段的"小盒子"原则（提醒）

V3 打完包后，开"编码 AI"的对话时：

1. 一次只开一个 Story 的对话
2. 系统消息塞 V3 中该 Story 包的全部上游文件 + V2 中该 Story 详情节
3. 用户消息只说："按 DoD 实现 S-xxx-yyy。完成后输出：① 代码 ② 自测对照表 ③ 偏离说明（若有）"
4. PR 描述必须链 V2 Story ID + V3 包 ID

> 这就是"丢一份小文档、AI 不会忘"的最终落地姿势。
