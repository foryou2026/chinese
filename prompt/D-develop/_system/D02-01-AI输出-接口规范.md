## AI 行为约束

1. page-id 映射为 URL + 定义 API 端点
2. **page-id 不增不减**：100% 来自 C02，需新增页面必须回 C02
3. **SM 转移全覆盖**：C02 每条状态转移都有接口承接
4. **入参/出参可溯源**：字段在 D01 实体中有对应
5. **错误码遵守 B01**
6. **不写 UI/视觉/HTML**
7. 未决项写末尾
