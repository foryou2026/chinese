## 文件 9：`prototype-style/` 可运行资产包

> **目的**：把 `design-system/` 中的 markdown 规范翻译成 C04 HTML 原型可以直接 `<link>` / `<script>` 的 CSS / JS 资产。**token 必须与 `design-system/01-tokens.md` 一字不差**。

### 9.1 `prototype-style/README.md`

```markdown
<!-- TARGET-PATH: docs/B04-design/prototype-style/README.md -->

# 原型样式包 · 使用说明

## 引入方式（C04 原型、相对路径引用，绝对不要拷贝）

C04 极简平铺：`<surface>/index.html` 与 `<surface>/<page-id>.html` 均位于 `docs/C04-prototype/<feature>/<surface>/`，深度统一 = docs 下 3 层，路径都是 `../../../B04-design/prototype-style/X`：

```html
<link rel="stylesheet" href="../../../B04-design/prototype-style/tokens.css">
<link rel="stylesheet" href="../../../B04-design/prototype-style/themes.css">
<link rel="stylesheet" href="../../../B04-design/prototype-style/app.css">
<script defer src="../../../B04-design/prototype-style/app.js"></script>
<script>proto.bootstrap();</script>
```

页面级微调样式 / 微交互一律内联在该 HTML 的 `<style>` / `<script>` 中；示例数据直接写在 HTML 静态片段，**不**产出 `mock-data.js` / `feature.css` / `feature.js`。例外：feature 若有 F4 上游资产，改用 `../../../../function/<feature>/ai/F4-AI-原型设计/_assets/X`（4 ups）。

## 全局命名空间

- CSS 变量：`--color-* / --space-* / --text-* / --radius-* / --shadow-* / --motion-* / --z-* / --font-*`
- JS 全局：`window.proto`，至少包含 `bootstrap()`、`switchTheme(mode)`、`switchAccent(name)`、`toast(...)`、`modal.open(...)`、`drawer.open(...)`

## DOM 三轴标记

| 轴 | 取值 | DOM |
|----|------|-----|
| 模式 | `light` / `dark` / `auto` | `<html data-mode="...">` |
| 主题色 | `<accent-1>` / `<accent-2>` / ... | `<html data-accent="...">` |
| 密度 | `default` / `compact` / `elder` | `<html data-density="...">` |

## 红线
1. 严禁在 feature.css / pages/*.html 中重定义任何 token；只可消费。
2. 严禁写死 hex / px。
3. 所有可聚焦元素必须可见焦点环（`--focus-ring`）。
4. 必须支持 `prefers-reduced-motion: reduce`。
```
