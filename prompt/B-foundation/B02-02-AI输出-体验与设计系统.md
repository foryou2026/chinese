## prototype-style/README.md

```markdown
# 原型样式包

## 引入方式

```html
<link rel="stylesheet" href="<相对路径>/prototype-style/app.css">
<script defer src="<相对路径>/prototype-style/app.js"></script>
<script>proto.bootstrap();</script>
```

页面级微调一律内联 <style>/<script>。

## 全局命名空间
- CSS：`--color-* / --space-* / --text-* / --radius-* / --shadow-* / --motion-* / --z-* / --font-*`
- JS：`window.proto`（bootstrap / switchTheme / toast / modal / drawer / dropdown）

## DOM 三轴标记

| 轴 | 取值 | DOM |
|----|------|-----|
| 模式 | light/dark/auto | `<html data-mode>` |
| 主题色 | accent-1/2/... | `<html data-accent>` |
| 密度 | default/compact/elder | `<html data-density>` |

## 红线
1. 严禁重定义 token
2. 严禁写死 hex/px
3. 可聚焦元素必须可见焦点环
4. 必须支持 prefers-reduced-motion: reduce
```

## 可运行资产说明

- `tokens.css`：把 `01-tokens.md` 所有 CSS 变量落到 `:root`，逐字一致
- `themes.css`：按 `07-responsive-dark.md` 给出暗黑/主题色/密度覆写
- `parts/*.css`：按组件域拆分（<=1200 行/文件），每组件实现 6 态+异常态
- `app.css`：@import 汇总
- `app.js`：`window.proto` 实现，纯 vanilla，零外部依赖
