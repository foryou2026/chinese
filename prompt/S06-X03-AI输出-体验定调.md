# 文案语气

## 整体调子
- 称呼：「你 / 您」
- 长度：<例：错误提示 ≤ 18 个汉字>
- 标点：<例：不用感叹号；不用问号"？"煽动>
- emoji：<允许 / 禁止 / 仅成功态>

## 对照表

| 场景 | 反例（我们不要） | 推荐 | 原因 |
|------|----------------|------|------|
| 加载中 | "正在飞速为您加载中..." | "加载中" | 不卖萌 |
| 提交成功 | "🎉 太棒了！" | "已保存" | 不感叹 |
| 必填校验 | "您似乎漏掉了什么呢？" | "请填写邮箱" | 直说 |
| 服务器错误 | "Oops!" | "服务器错误，已记录。请稍后再试。" | 给下一步 |
| 空数据 | "这里空空如也" | "暂无数据。点击上方按钮新建。" | 给路径 |

## 多语言策略（如需）
- 翻译来源：人工 / AI / 双轨
- 关键词术语表位置：
