# 文案语气

> **直说，不卖萌。给下一步，不光报错。**

---

## 一、整体调子

| 维度 | 规则 |
|------|------|
| 称呼 | 默认「你」；面向高龄用户或正式场合切「您」（由 `data-locale-formal` 控制，全局统一，不混用） |
| 长度 | 错误提示 ≤ 18 个汉字；按钮文字 ≤ 6 个汉字；标题 ≤ 12 个汉字 |
| 标点 | **禁用感叹号 `!`**；**禁用煽动性问号 `？`**；句末"。"可省略；分句用全角逗号 |
| Emoji | **全局禁用**（功能控件 / 文案 / 标题一律不出现） |
| 数字 | 必须 `tabular-nums`；金额 `¥1,234.00`；时间 `YYYY-MM-DD HH:mm`；时长 `1h 23m` 而非 `1小时23分钟` |
| 字体 | 文案默认 `--font-sans`；正式场景标题用 `--font-display`；行楷仅品牌名 |
| 大小写（英文） | 句首大写，按钮 Title Case，导航 Title Case，正文 Sentence case |

---

## 二、5 类典型场景对照表

| 场景 | 反例（不要） | 推荐 | 原因 |
|------|-------------|------|------|
| 加载中 | 「正在飞速为您加载中…」 | 「加载中」 | 不卖萌；不承诺速度 |
| 提交成功 | 「🎉 太棒了！保存成功！」 | 「已保存」 | 不感叹；不 emoji |
| 必填校验 | 「您似乎漏掉了什么呢？」 | 「请填写邮箱」 | 直说漏在哪 |
| 服务器错误 | 「Oops! 出错啦~」 | 「服务暂不可用，已记录。请稍后重试或联系管理员。」 | 给下一步 + 安抚（已记录） |
| 空数据 | 「这里空空如也」 | 「暂无数据。点击右上角「新建」开始。」 | 给路径 |

## 三、扩展场景对照

| 场景 | 推荐文案 |
|------|---------|
| 危险操作二次确认（删除） | 标题：「删除「{name}」？」 / 描述：「删除后无法恢复。」 / 按钮：取消 \| 确认删除（朱砂） |
| 离开未保存 | 标题：「离开此页？」 / 描述：「未保存的修改将丢失。」 / 按钮：留下 \| 离开 |
| 权限不足 403 | 标题：「无权访问」 / 描述：「当前角色不具备此操作权限。请联系管理员。」 |
| 网络中断 | Toast：「网络已断开，正在重连…」（不要表情符号） |
| 表单字段帮助 | `--gray-500` 13px，紧贴输入框下方；句末不加句号 |
| Tooltip | ≤ 14 个汉字，纯陈述句 |
| 操作进行中按钮 | 「保存中…」（同色 disabled，不换文案为「请稍候」） |
| 批量操作 | 「已选 {n} 项 · 批量删除」（数字必须 `tabular-nums`） |
| 时间相对值 | ≤ 1 分钟「刚刚」；< 60 分钟「{n} 分钟前」；< 24h「{n} 小时前」；其他「YYYY-MM-DD」 |
| 价格 | 「¥1,234.00」（千分位 + 2 位小数 + 全角¥需为半角） |

## 四、错误信息结构铁律

任何错误提示必须 = **「现象」+「原因（如果用户能理解）」+「下一步」**：

```
✗ 「保存失败」
✗ 「Error: 500」

✓ 「保存失败：网络中断。请检查网络后重试。」
✓ 「无法导入：第 12 行缺少「邮箱」列。请补全后重新上传。」
```

---

## 五、多语言策略（如需）

- 默认中文；按需扩展英文（`en-US`），不做繁体（统一简体）。
- 翻译来源：人工 + AI 双轨，由产品最终审定。
- 术语表落盘：`docs/A00-meta/glossary.md`。
- UI 字符串集中：`/i18n/zh-CN.json`、`/i18n/en-US.json`，不允许内联字符串。

---

## 六、AI 生成内容标识（如有）

任何由 AI 生成的内容（建议、自动摘要、对话回复）必须：

- 在容器右上角附 `🤖`-style 文字角标（**注意**：用文字「AI 生成」而非 emoji，符合"禁用 emoji"原则）；
- 紧随容器下方附统一组件 `<DisclaimerAI/>`：
  > 「本内容由 AI 辅助生成，请人工复核后采用。」
- 若涉及决策建议，必须可被一键标记「AI 误差」回流改进。
