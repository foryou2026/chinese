## 0. 摘要（≤ 5 行）

1. 沉静、克制、有气韵的中国风现代后台体验，而非"医院 HIS / SaaS 模板 / 国风装饰"任一刻板印象。
2. 视觉锚点：宣纸米底 + 宋瓷墨青 + 全域毛玻璃 + 行楷大字 + Three.js 流体波浪。
3. 5 条不可逾越保证调性不漂移；6 条原则保证决策一致。
4. 文案"直说不卖萌"、错误"给下一步"、空态"给路径"。
5. 默认主题为亮色"宣纸"，标准支持"墨夜"暗色 + 5 套主题色（墨青 / 朱砂 / 翠玉 / 鎏金 / 古墨）切换。
