# 一页纸定调

> **阶段**：X · 体验总监
> **上游**：默认规范
> **冻结状态**：已冻结
> **下游**：S/I/N/H 全部硬约束

---

## 0. 摘要

沉静、克制、有气韵的中国风现代后台体验。
不是装饰国风，不是冰冷工程台，不是 SaaS 模板。

---

## 1. 一句话定调

> **「像一张温润的宣纸，铺开宋瓷墨青色的信息，再覆一层薄玻璃。」**
>
> 它必须**沉得住气**（不躁），**留得住人**（不冷），**说得清话**（不绕）。

---

## 2. 5 条不可逾越（一票否决）

| # | 条目 | 可验证维度 |
|---|------|-----------|
| 1 | 禁止纯白 / 纯黑大面积色块 | 亮色页面背景必须使用宣纸米底渐变 token；正文字色必须使用 `--gray-900: #1F1A14` 暖墨而非 `#000` |
| 2 | 禁止"国风装饰"陷阱 | 不出现卷轴杆 / 大幅毛笔水墨字背景 / 印章图章替代 logo / 楷草隶大字铺底 |
| 3 | 禁止业务页正文使用行楷字体 | 仅品牌名、登录/欢迎页 Hero H1、印鉴 `.seal` 三处允许 `--font-brush`；表格、表单、列表必须 `--font-sans` |
| 4 | 禁止动效 > 300ms / 禁止视差滚动 | 所有过渡 ≤ 250ms；缓动统一 `cubic-bezier(.2,0,0,1)`；`prefers-reduced-motion: reduce` 时全部关闭 |
| 5 | 禁止 hex 硬编码 | 所有颜色必须走 CSS 变量；按钮 / 链接 / 焦点环必须随 `data-accent` 自动切换 |

> 任何 PR / 设计稿命中以上任一条 → **打回，不讨论**。

---

## 3. 5 条主张（鼓励 / 优先做）

| # | 主张 | 可验证维度 |
|---|------|-----------|
| 1 | 信息密度优先于呼吸感 | 默认表格行高 44px、表单字号 17px、卡片 padding 22~24px；不为"留白美学"牺牲一屏信息量 |
| 2 | 玻璃 + 毛玻璃为唯一表面语言 | TopBar / 卡片 / 按钮 / 下拉 / 模态 / 标签 必须 `backdrop-filter: blur(22~34px)`；输入框内底允许实色保证可读 |
| 3 | 数字必须 `tabular-nums` | 表格、统计、金额、时间戳、ID 一律启用 `font-variant-numeric: tabular-nums`，避免错位跳动 |
| 4 | 焦点环必须 4px 青花光晕 | `:focus-visible` 统一 `box-shadow: 0 0 0 4px rgba(46,92,138,.18)`；键盘可达 100% 主流程 |
| 5 | 错误 / 空态 / 加载 必须给"下一步" | 不允许出现"加载中…"或"暂无数据"光秃秃的占位，必须 1 行文案 + 1 个主操作按钮 |

---

## 4. 检验标准（1 分钟判定）

把任何设计稿往下面 6 题里套，全部回答 ✅ 才算通过：

- [ ] 截图缩到 30%，能否一眼识别这是"宣纸 + 墨青 + 玻璃"而非通用 SaaS？
- [ ] 把所有色值复制出来，是否全部能在 `tokens.css` 里找到对应变量？
- [ ] 把 `prefers-reduced-motion: reduce` 打开，页面是否仍然可用且无闪烁？
- [ ] 切换 `data-mode="dark"`，对比度是否依然 WCAG AA、行楷大字是否依然可读？
- [ ] 切换 `data-accent="cinnabar"`，主按钮 / 链接 / 焦点环是否同步换色，**且不出现孤岛色**？
- [ ] 主操作按钮每页 ≤ 1 个？
