# 课程知识点调研结论

> 原型文件: [demo.html](./demo.html) — 浏览器打开即可查看全部10种知识点的应用端+管理端原型

## 知识点类型总览

| # | 类型 | 内容量级 | AI依赖 | 核心交互 |
|---|------|---------|--------|---------|
| 1 | 拼音 | 声母23+韵母24+整体认读16 | 无 | 听音→辨音→声调训练→拼读 |
| 2 | 偏旁部首 | ~214个常用部首 | 无 | 认部首→理解含义→拆解汉字 |
| 3 | 笔画笔顺 | 基本笔画~30种 + 常用字笔顺 | 无 | 观看动画→描红练习(Canvas) |
| 4 | 汉字 | HSK1-6约2600个 | 无 | 字卡学习→组词→例句→听写 |
| 5 | 词语 | HSK1-6约5000+ | 无 | 词卡→例句→分类记忆 |
| 6 | 成语 | 常用~500个 | 无 | 四字展示→字面义→引申义→典故→例句 |
| 7 | 短句 | 按主题场景,每主题10-30句 | 无 | 逐词点击→整句听读→语法标注 |
| 8 | 语法句型 | HSK1-6约300+语法点 | 无 | 模板展示→色彩标注→例句池→替换练习 |
| 9 | 对话场景 | 按场景分类,每场景5-15轮 | 角色扮演评分需AI(高级) | 气泡对话→逐句听读→角色扮演 |
| 10 | 阅读理解 | 按HSK分级,每级50-100篇 | 无 | 分级阅读→点词查义→生词标注 |

## 核心战略对齐："一次生成，无限分发"

**10种知识点类型中，9种完全不需要实时AI调用。** 所有内容（文本、音频、动画、标注）均可在后台预生成后静态分发：

- **文本数据**: 拼音、汉字、词语、成语等结构化数据，批量导入
- **音频数据**: TTS批量生成或人工录制，存CDN分发
- **动画数据**: 笔顺SVG/Lottie预渲染，前端播放
- **交互判定**: 描红用Canvas+DTW算法前端对比，不调AI

唯一需要AI的场景：**对话场景中的角色扮演语音评分**，归类为高级用户功能。

## 管理端核心配置模式

所有知识点类型遵循统一配置范式：
1. **基础字段**: 内容本体 + 拼音 + 中英释义
2. **媒体资源**: 音频上传/TTS + 图片/动画
3. **分类标签**: HSK等级 + 主题 + 难度 + 排序
4. **关联关系**: 知识点之间的引用关联
5. **批量操作**: CSV导入 + AI辅助批量生成

| 维度 | Duolingo | HelloChinese | 我们的定位 |

| SRS复习 | 无正式SRS | 有 | 需设计(非本次范围) |
| 分级阅读 | 有Stories | 有分级故事 | 有,支持点词查义 |
