# 参考清单

> 至少 3 个。每个参考都明确"借什么、不借什么"。

---

## REF-1：Linear（linear.app）

- **借这一点**：键盘可达 100% 主流程；命令面板 (`Cmd+K`) 即时反馈；状态色克制（仅紫一色 + 中性灰阶）；表格 `tabular-nums` 数字对齐。
- **不借**：纯暗色冷工程感；几何字标 / 机械感品牌；零装饰的"工具属性"。
- **白描**：左侧窄侧栏 + 中间长列表 + 右侧轨道详情；hover 浮起 1px；圆角 6~8px；阴影几乎不可见。

## REF-2：Apple Human Interface Guidelines · Vision Pro Glass

- **借这一点**：玻璃面板的"近实远虚"层级感；`backdrop-filter` 的边缘高光（顶部 1px 暖白渐变）；玻璃投影的"墨色淡阴影"。
- **不借**：3D 空间感 / 深度推拉动效；圆角全 999px 的胶囊化。
- **白描**：每个玻璃面板顶部都有一道极细的白色高光带，模拟釉光；面板之间用模糊度差异（22px vs 34px）拉开远近。

## REF-3：宋代汝窑天青釉 · 故宫博物院藏品

- **借这一点**：主色 `#1B3A5C`（宋瓷墨青）的明度饱和度区间；釉面的"半透 + 微反光"质感映射到玻璃材质 `clearcoat: 0.6`；"开片纹"启发 1px 浅描边样式。
- **不借**：器物本身造型 / 任何 3D 还原；古董色的褪色感（我们要新瓷釉光）。
- **白描**：整体色调温润但**冷静偏蓝**，不是水墨灰，也不是孔雀绿。

## REF-4：明代宣德宣纸（暖米色背景）

- **借这一点**：背景渐变 `#F8F2E0 → #F2EBD3 → #E8EFF5`（老宣纸 → 米色 → 月白瓷）；不刺眼、不冷漠；正文长时间阅读不疲劳。
- **不借**：宣纸本身的褶皱 / 纹理大量铺底（最多 18% opacity 的噪点 `paper-grain`）。

## REF-5：Notion · 表格与数据库视图

- **借这一点**：表格 hover 行的极轻染色 `rgba(207,221,232,0.45)`；可拖拽列宽；空态卡的"插画 + 1 行说明 + 1 个按钮"模式。
- **不借**：满屏 emoji；无品牌色感的纯灰白；过度依赖 / 命令；自由编辑器的歧义性。

## 共同模式：`借` 的可被自动验证

| 借鉴维度 | 我们的具体值 | 来源 |
|---------|------------|------|
| 表格数字对齐 | `font-variant-numeric: tabular-nums` | Linear / Stripe |
| 玻璃面板顶高光 | `inset 0 1px 0 rgba(255,250,235,.65)` | Apple visionOS |
| 主色色相 | HSL(212, 53%, 23%) ≈ `#1B3A5C` | 宋瓷天青 |
| 背景温度 | 暖米 → 冷月白渐变 | 宣纸 |
| 焦点环 | `0 0 0 4px rgba(46,92,138,.18)` | Stripe / Linear |
