## Stage 1：入门 Starter（对标 HSK 1）

### 内容标准：100% HSK 官方大纲

Stage 1 所有词汇、语法、语音内容严格来自 HSK 1 级官方大纲，不增不减。

### 整体指标

| 指标 | 数值 |
|------|------|
| 章数 | 3章 |
| 每章节数 | 4节（3学习节 + 1练习节） |
| 总节数 | 12节 |
| 新词汇 | 500词（HSK 1 词表） |
| 新汉字 | 300字（认读）/ 120字（书写） |
| 新语法点 | 30个 |
| 预计总学时 | 30-40小时 |
| 每日15分钟 | 约3个月 |

### 启用的课程 L2 学习形式（10/18种）

| L1知识类型 | 启用的L2 | 说明 |
|----------|---------|------|
| 拼音 | 声韵母学习 + 声调训练 + 拼读组合表 | 全部3种，集中在Ch1完成 |
| 偏旁部首 | 部首卡片 | 引入50个高频部首 |
| 笔画笔顺 | 笔画认知 + 笔顺描红 | 8种基本笔画 + 基础字描红 |
| 汉字 | 字卡学习 | 基础汉字卡片 |
| 词语 | 词卡学习 | HSK 1词表词卡 |
| 短句 | 短句学习 | 基础问候/介绍/数字短句 |
| 语法句型 | 句型模板 | 基础句型（是字句/有字句等） |

### 启用的考试 L3 变体（11种免费 + 2种付费）

**免费**：汉字选拼音, 听音选拼音, 拼音选汉字, 释义选汉字, 词义选择, 汉字写全拼, 声调标注, 汉字↔拼音配对, 拼音正误, 声调正误, 图选汉字

**付费**：单字跟读, 词语跟读
