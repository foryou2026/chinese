# 测试体系与记忆曲线策略

> 节测试 / 章测试 / 阶段测试 / 错题集 / 间隔复习 · 全部开放无门槛

---

## 一、核心原则：开放测试，客观记录

**所有测试均为开放性评估，不设强制解锁门槛。**

- 用户可自由跳到任何阶段学习，测试结果仅作为能力评估和学习进度的客观记录
- 测试通过/未通过不会阻断用户的学习路径
- 系统根据测试结果提供推荐建议（如"建议先巩固 Stage 2 内容"），但用户有完全自由跳过

---

|------|-------|-------|---------|----------|

| **考察范围** | 仅本节知识点 | 本章全部知识点 | 本阶段全部+前置回顾 | 仅错题 |

## 三、节测试(Section Test)详细设计

### 3.1 题量与构成

每节测试固定 **10道题**，从该节涉及的知识点中抽题：

| 题型(L1) | 题数 | 出题策略 |
|----------|------|---------|
| 选择题 | 3-4题 | 覆盖本节核心词汇/语法，优先使用高频L3变体 |
| 配对题 | 1-2题 | 汉字↔拼音 或 词语↔释义 |
| 填空题 | 2-3题 | 拼音填写 或 语境补词 |
| 判断题 | 1-2题 | 拼音正误 或 词义正误 |
| 排序题 | 0-1题 | 词语排句（Stage 2+才引入） |

### 3.2 各阶段节测试的 L3 变体选择

| 阶段 | 可用L3变体（节测试用简单、快速的） | 回顾比例 |
|------|-------------------------------|---------|
| Stage 1 | 汉字选拼音, 听音选拼音, 拼音选汉字, 释义选汉字, 词义选择, 汉字写全拼, 声调标注, 汉字↔拼音, 拼音正误, 声调正误, 图选汉字 | 100%本节 |
| Stage 2 | +听音辨调, 语境选词, 释义选词, 语法词填选, 词义正误, 翻译正误, 词语↔释义, 词语排句, 英中/中英选择 | 80%本节+20%回顾 |
| Stage 3 | +成语选义, 含义选成语, 选正确句, 语法词填写, 翻译填词, 成语↔含义, 对话排序, 句子正误 | 70%本节+30%回顾 |
| Stage 4 | +同义词辨析, 听音选词, 错句辨别, 语境选成语, 量词填写, 用法正误 | 60%本节+40%回顾 |
| Stage 5-6 | 所有66种免费L3均可选用 | 50%本节+50%回顾 |

### 3.3 出题逻辑（题库抽取规则）

```
节测试出题引擎:
1. 确定本节的知识点集合 K = {k1, k2, ..., kn}
2. 从K中按权重抽取核心知识点（新词汇 > 新语法 > 新汉字）
3. 为每个知识点匹配适合的L1题型 → L2知识类型 → L3变体
   （必须从该阶段已启用的L3变体中选取）
4. 如有回顾比例，从该章前序节或前序章的低掌握度知识点中补充
5. 打散题目顺序，确保不连续出现同一题型
6. 生成选项干扰项（形近字/同音字/近义词/语法混淆项）
```

## 五、阶段测试(Stage Test)详细设计

### 5.1 题量与构成

| 阶段 | 题数 | 时间 | 参考标准 | 说明 |
|------|------|------|---------|------|
| Stage 1 | 60题 | 45分钟 | 70%=42/60 | 基础能力评估 |
| Stage 2 | 65题 | 50分钟 | 70%=46/65 | 评估后推荐主题方向 |
| Stage 3 | 70题 | 55分钟 | 70%=49/70 | 含行业专属内容 |
| Stage 4 | 75题 | 55分钟 | 70%=53/75 | 题目难度显著提升 |
| Stage 5 | 80题 | 60分钟 | 70%=56/80 | 含长阅读理解 |
| Stage 6 | 80题 | 60分钟 | 70%=56/80 | 最终能力认证 |

> **全部阶段测试均不阻断学习进度**。测试结果记录到用户档案，系统给出能力评估和学习建议。

### 5.2 阶段测试模块划分

以 **Stage 4 阶段测试(75题/55分钟)** 为例：

| 模块 | 题数 | 时间 | 内容 |
|------|------|------|------|
| A：词汇与汉字 | 20题 | 12分钟 | 选择题+填空题+配对题，考察词义/拼音/汉字 |
| B：语法与句型 | 15题 | 10分钟 | 选择题+填空题+排序题，考察语法运用 |
| C：听力理解 | 15题 | 15分钟 | 听力复合题+听写，含对话和段落 |
| D：阅读理解 | 15题 | 12分钟 | 阅读选择+判断+翻译 |
| E：综合运用 | 10题 | 6分钟 | 成语+对话排序+看图选词 |

### 5.3 阶段测试出题范围

```
阶段测试知识点来源:
├── 本阶段全部知识点      ← 60% 权重
├── 前一阶段核心知识点    ← 25% 权重（验证长期记忆）
└── 更早阶段的高频错点    ← 15% 权重（消灭顽固弱点）
```

### 5.4 测试结果处理（建议制，非强制）

```
阶段测试结果处理:

✅ 通过 (≥70%):
├── 记录评分到用户档案
├── 颁发阶段徽章（成就系统）
└── 推荐进入下一阶段（用户本来就可以自由进入）

⚠️ 未通过 (<70%):
├── 生成《薄弱分析报告》
│   ├── 按知识类型统计正确率
│   ├── 标记最薄弱的3-5个知识点
│   └── 推荐针对性复习路径
│
├── 提供定向复习模式（建议，不强制）
│   ├── 自动生成薄弱点的额外练习
│   └── 完成后建议重新参加测试
│
├── 重考规则
│   ├── 不限重考次数
│   ├── 每次重考从题库重新抽题（非原题）
│   └── 两次测试间隔至少2小时（防止刷记忆答案）
│
└── ❗ 不阻断学习路径
    └── 用户仍可自由进入任何阶段学习
```

## 六、各层测试的 L3 变体使用策略

### 6.1 节测试（简单、快速、即时反馈）

| 知识类型 | 推荐L3变体 | 原因 |
|---------|----------|------|
| 拼音 | 汉字选拼音, 听音选拼音, 拼音正误, 声调正误 | 最直观的认知检验 |
| 汉字 | 拼音选汉字, 释义选汉字, 汉字↔拼音配对 | 快速识别 |
| 词汇 | 词义选择, 语境选词, 词语↔释义配对 | 核心语义理解 |
| 语法 | 语法词填选, 选正确句 | 即时语法应用 |
| 成语 | 成语选义, 含义选成语 | 基础理解 |

### 6.2 章测试（全面、综合）

在节测试基础上增加：
- 填空题：语境补词, 翻译填词, 语法词填写
- 排序题：词语排句, 对话排序
- 听力题：听句选答, 听词选义
- 翻译题：英中选择, 中英选择
- 听写题：听写词语
- 看图选词：图选词语, 图选描述句

### 6.3 阶段测试（高难度、综合能力）

在章测试基础上增加：
- 复合听力：短对话选择, 长段落选择, 听对话答题
- 高级填空：听音写拼音, 量词填写, 偏旁补全
- 高级翻译：英中输入, 中英输入
- 高级听写：听写短句, 听写段落
- 综合运用：看图选词全部4种变体, 成语↔含义配对, 句子排段

## 七、错题集(Wrong Answer Collection)系统

### 7.1 错题收录规则

| 来源 | 收录条件 | 收录信息 |
|------|---------|---------|
| 节测试 | 答错即收 | 原题+正确答案+用户答案+涉及知识点+错误时间 |
| 章测试 | 答错即收 | 同上 |
| 阶段测试 | 答错即收 | 同上 |
| 练习节 | 答错即收 | 同上 |

### 7.2 错题状态机

```
错题生命周期:
┌──────────┐    连续答对2次     ┌──────────┐    连续答对3次     ┌──────────┐
│  新错题   │ ────────────────→ │  复习中   │ ────────────────→ │  已掌握   │
│  (Red)   │                   │ (Yellow) │                   │ (Green)  │
└──────────┘                   └──────────┘                   └──────────┘
      ▲                              │                              │
      │          再次答错             │          再次答错             │
      └──────────────────────────────┘──────────────────────────────┘
```

- **新错题(Red)**：刚答错，需要尽快复习
- **复习中(Yellow)**：已答对2次，但还需巩固
- **已掌握(Green)**：连续答对3次（间隔至少1天），从活跃错题中移出
- **任何阶段再答错**：立即回退到"新错题"

### 7.3 错题练习模式

| 模式 | 说明 | 题量 |
|------|------|------|
| 快速练习 | 随机10-20题 Red/Yellow 错题 | 10/20题可选 |
| 知识类型专练 | 按知识类型筛选（如"语法错题"） | 10-30题 |
| 章节专练 | 按章节筛选错题 | 该章全部错题 |
| 高频错题 | 答错次数最多的 Top 20 | 20题 |
| 错题考试 | 正式计时测试，用错题组卷 | 用户选10/20/30/50题 |

## 八、记忆曲线(Spaced Repetition)复习策略

### 8.1 间隔复习时间表

综合 SM-2 算法、Leitner Box 和移动端场景，采用 **1-3-7-14-30** 核心节奏：

| 复习轮次 | 间隔 | 触发时机 | 预期保持率 | 复习形式 |
|---------|------|---------|----------|---------|
| R0 | 即时 | 学习节结束时 | ~100% | 节测试(10题) |
| R1 | 1天后 | 次日推送 | ~33%→恢复到~85% | 迷你复习(5题) |
| R2 | 3天后 | 第3天推送 | ~60%→恢复到~90% | 迷你复习(5题) |
| R3 | 7天后 | 第7天推送 | ~70%→恢复到~92% | 迷你复习(5题) |
| R4 | 14天后 | 第14天推送 | ~80%→恢复到~95% | 迷你复习(5题) |
| R5 | 30天后 | 第30天推送 | ~85%→恢复到~97% | 章测试(如恰好完成章) |
| R6 | 60天后 | 第60天推送 | ~90%→近永久 | 融入阶段测试 |

### 8.2 迷你复习(Mini Review)设计

每次仅 **5题 / 2-3分钟**，是记忆曲线的核心执行单元：

```
迷你复习触发逻辑:
1. 系统每天检查: 有哪些知识点到了复习时间点？
2. 按优先级排序: Red错题 > 新学(R1) > R2到期 > R3到期 > ...
3. 从到期知识点中抽取5个，生成5道题
   （题型从该阶段已启用的L3变体中选取）
4. 用户答题:
   ├── 答对 → 该知识点进入下一个间隔(如R1→R2)
   └── 答错 → 重置为R1(1天后再复习) + 加入错题集
5. 日推送策略:
   ├── 早上: 推送昨天R1到期的知识点
   ├── 傍晚: 推送R2/R3到期的知识点
   └── 每日上限: 最多3组迷你复习(15题)，避免复习疲劳
```

### 8.3 SM-2 自适应难度算法

```
核心公式:
  next_interval = current_interval × ease_factor

ease_factor 调整规则:
  初始值: 2.5
  答对(Easy):  ease_factor += 0.15  (最大3.0)
  答对(Good):  ease_factor 不变
  答对(Hard):  ease_factor -= 0.20  (最小1.3)
  答错:        ease_factor -= 0.30  (最小1.3), 重置interval=1天

示例:
  知识点"把"字句, ease_factor=2.5
  R1=1天后复习, 答对(Good) → R2 = 1 × 2.5 = 2.5天 ≈ 3天
  R2=3天后复习, 答对(Easy) → R3 = 3 × 2.65 = 7.9天 ≈ 8天
  R3=8天后复习, 答错 → 重置! R1=1天后, ease_factor降为2.2
```

### 8.4 复习与新课的平衡

| 场景 | 每日推荐分配 | 时间 |
|------|------------|------|
| 正常学习日 | 1节新课(8-15m) + 1-2组迷你复习(4-6m) | 12-21分钟 |
| 复习积压日 | 0节新课 + 3组迷你复习(9m) | 9分钟 |
| 测试日 | 章/阶段测试 | 15-60分钟 |
| 错题强化日 | 错题专练20题 | 10-15分钟 |

**核心原则**：复习优先于新课。如果当日到期复习量超过15题，建议暂停新课学习，优先完成复习（建议制，不强制）。

## 十一、关键指标与健康度监控

### 11.1 用户学习健康指标

| 指标 | 健康范围 | 预警值 | 干预策略 |
|------|---------|-------|---------|
| 每日学习时长 | 10-30分钟 | <5分钟 | 推送鼓励通知 |
| 节测试通过率 | ≥70% | <50% | 建议降速复习 |
| 复习完成率 | ≥80% | <50% | 建议暂停新课，优先复习 |

| 连续学习天数 | ≥3天/周 | 断学>3天 | 推送召回通知+降低当日目标 |
