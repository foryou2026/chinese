# HSK 各级别能力描述

> 来源：HSK3.0考试能力描述.docx

HSK (中文水平考试) 是一项国家级标准化考试，重点考查中文作为第二语言的考生在生活、学习和工作中运用中文进行交际的能力。

## 能力等级

| 级别 | 能力描述 | 交际场景 | 关键词 |
|------|---------|---------|--------|
| HSK 1级 | 能用中文就生活场景进行简单交流 | 生活 | 简单 |
| HSK 2级 | 能用中文就生活、学习、工作场景进行基本交流 | 生活/学习/工作 | 基本 |
| HSK 3级 | 能用中文就生活、学习、工作场景进行有效交流 | 生活/学习/工作 | 有效 |
| HSK 4级 | 能用中文就生活、学习、工作场景进行完整、连贯交流 | 生活/学习/工作 | 完整、连贯 |
| HSK 5级 | 能用中文就工作、学业场景进行准确、恰当交流 | 工作/学业 | 准确、恰当 |
| HSK 6级 | 能用中文就职场、学业、一般专业场景进行丰富、顺畅交流 | 职场/学业/一般专业 | 丰富、顺畅 |
| HSK 7级 | 能用中文就职场、一般专业场景、通用学术场景进行规范、流利交流 | 职场/专业/学术 | 规范、流利 |
| HSK 8级 | 能用中文就职场、一般专业场景、通用学术场景进行得体、深入交流 | 职场/专业/学术 | 得体、深入 |
| HSK 9级 | 能用中文就职业、专业、学术等各种场景进行精准、深入交流 | 全场景 | 精准、深入 |

## 软件设计参考

- **1-3级**：初级阶段，UI需大量拼音辅助，图片支撑理解，词汇以日常生活为主
- **4级**：过渡阶段，逐步去拼音，增加段落级阅读，引入简单写作
- **5-6级**：高级阶段，纯汉字，长篇文章，高级写作与表达
- **7-9级**：专业/学术阶段（当前考试样卷仅覆盖1-6级笔试 + 3-6级口语）

## 拼音辅助策略

| 级别 | 拼音出现方式 |
|------|-------------|
| 1级 | 全文拼音标注（题干+选项均有拼音） |
| 2级 | 全文拼音标注 |
| 3级 | 部分拼音（阅读第一部分有拼音，其余纯汉字） |
| 4级+ | 无拼音（纯汉字） |
