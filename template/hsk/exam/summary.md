## 题型全景速查

| 技能 | 题型关键词 | 覆盖级别 | 交互形式 |
|------|-----------|---------|---------|
| 听力 | 听句选图 | 1-3级 | 听音频 → 从图片中选择 |
| 听力 | 听问题选词语 | 1级 | 听问题 → 选带拼音词语 |
| 听力 | 听对话选图 | 1-3级 | 听对话 → 匹配图片 |
| 听力 | 听段落选答案 | 1-6级 | 听段落/独白 → 四选一 |
| 听力 | 听对话选答案 | 2-6级 | 听对话 → 四选一 |
| 听力 | 听新闻判断 | 6级 | 听短文 → 判断主旨/观点 |
| 阅读 | 拼音句配图 | 1-2级 | 看拼音句子 → 选图片 |
| 阅读 | 拼音句填词 | 1-2级 | 看拼音句子 → 从选项选词填空 |
| 阅读 | 句子配对 | 2-3级 | 问句↔答句 / 上下句匹配 |
| 阅读 | 选词填空 | 3-6级 | 从词库中选词填入句子/段落 |
| 阅读 | 段落阅读理解 | 3-6级 | 阅读段落 → 选择题 |
| 阅读 | 语句填空 | 4-6级 | 段落空缺处 → 选择合适句子填入 |
| 阅读 | 段落排序 | 5级 | 打乱段落 → 重新排列正确顺序 |
| 书写 | 看拼音写汉字 | 2-3级 | 看拼音提示 → 填写汉字 |
| 书写 | 拼音组词排序 | 2级 | 打散拼音词卡 → 拖拽排句 |
| 书写 | 看图写句 | 3-5级 | 看图片+关键词 → 写句子 |
| 写作 | 命题短文 | 4-6级 | 根据题目要求 → 写短文/长文 |
| 写作 | 看图写作 | 5级 | 看4张图 → 写故事（≥100字） |
| 写作 | 应用文写作 | 6级 | 场景要求 → 写招租/通知等 |
| 写作 | 议论文写作 | 6级 | 话题 → 写观点论述（≥300字） |
| 口语 | 听后重复 | 3级 | 听句子 → 原样复述 |
| 口语 | 听后复述 | 4-6级 | 听段落 → 用自己的话复述 |
| 口语 | 看图说话 | 3-6级 | 看图片 → 口头描述 |
| 口语 | 回答问题 | 3-6级 | 看/听问题 → 口头回答 |
