## 地域聚焦：东南亚本地生活

- **目标城市**：曼谷/清迈（泰国）、胡志明市/河内（越南）、雅加达/巴厘岛（印尼）
- **生活场景**：中国外派人员、留学生、旅居者、数字游民在东南亚的日常生活
- **语言场景**：与本地人用中文交流日常事务（东南亚华人社区/中国城/中文服务机构）
- **文化要素**：热带气候生活适应、东南亚饮食文化（冬阴功/河粉/沙爹）、佛教文化、宋干节/开斋节、摩托车出行文化

## 语法对齐说明

本主题96个语法点与HSK3级完全一致，但所有例句和示范内容全部替换为东南亚日常生活场景。例如：
- HSK原句「你把箱子放在这儿吧」→ 日常「你把行李放在公寓里吧」
- HSK原句「如果明天不下雨，我们就去爬山」→ 日常「如果明天不下雨，我们就去夜市逛逛」
- HSK原句「只有努力学习，才能得到好成绩」→ 日常「只有学会一些泰语，才能更好地融入本地生活」
