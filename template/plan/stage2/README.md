## HSK2 新增语法分类索引

### 动作的态（3项）
- 持续态：动词+着（动作持续/状态持续）
- 经历态：动态助词"过"

### 句子成分（8项）
- 受事主语
- 动词/动词性短语/主谓短语作定语
- 数量补语2（形容词+数量补语 / 动作持续时间）
- 状态补语1（动词+得+形容词性短语）
- 结果补语1（动词+错/懂/好/会/完）
- 趋向补语1（简单趋向 / 复合趋向）
- 名词性短语作谓语
- 主谓谓语句

### 句子类型（15项）
- 选择问句（还是）
- 用"别"的祈使句
- 因果复句、转折复句、选择复句、紧缩复句
- 兼语句1、双宾语句2、存现句2
- "是"字句2、"是……的"句1
- "有"字句2、比较句1（4种格式）

### 词类（17项）
- 动词重叠、形容词重叠
- 离合词（帮忙/游泳/跳舞/跑步）
- 能愿动词（可能）
- 多种介词（比/给/跟/从/往/因为）
- 多种副词（别/还/已经/快/最/一起/经常等）
- 结构助词（得/地）
- 新量词（条/位/间/名/次/包）

### 固定格式（3项）
- 要/快要/就要……了
- 还是……吧
- 都……了

### 短语（7项）
- 动词性/名词性/形容词性短语
- 固定短语（什么的）
- 兼语短语、"的"字短语、连谓短语、动补短语

### 特殊表达法（4项）
- 序数表示法2
- 概数表达法1（三种格式）

### 语素（1项）
- 后缀 —面
