## HSK3 新增语法分类索引

### 句子成分（11项）
- 主语：主谓短语作主语 / 动词性/形容词性短语作主语
- 定语：多项定语
- 补语：可能补语1 / 数量补语3 / 程度补语1(极了/坏了/得很) / 结果补语2 / 趋向补语2(引申用法：开始/持续/结果)

### 句子类型（27项）
- 句类：反问句1(不是……吗) / 特指问句(怎样+动词)
- 复句：假设(如果/的话) / 并列(一会儿/一边/又) / 承接(先……再/然后) / 条件(只有/只要) / 目的(为了) / 紧缩(了就) / 转折(虽然可是) / 选择(或者) / 递进(不但而且)
- 特殊句型：存现句3 / 把字句1(3种格式) / 是的句2 / 比较句2(5种格式) / 被动句1(2种格式) / 连动句2 / 重动句

### 固定格式（11项）
- X什么(啊) / 一……也/都+不/没 / 一点儿也不 / 从……起 / 在……上/下/中 / 以前/以后 / 对……来说 / 该……了 / 越……越 / 除了……还/也/都

### 词类（32项）
- 介词：根据 / 为/向/关于 / 向(方向) / 把/被 / 为了/为
- 代词：别人/咱们 / 别的/其他 / 怎样 / 疑问代词非疑问用法(不定指/任指)
- 副词：程度(比较/还/更/特别/挺/有些/越/极) / 时间(才/马上/先/一会儿/刚/刚刚/一直) / 范围(就/一块儿/一共/只/到处/只是) / 语气(当然/其实/终于/才/就) / 频率(总/总是/又/常常) / 关联(一边) / 否定(不必/不用) / 情态(大概/必须/差不多/一定/好像/几乎)
- 动词：离合词(动宾式/动补式) / 能愿动词(需要/该/应该/愿意/得)
- 名词：方位名词(东/南/西/北/北方/东方/南方/西方/中间)
- 量词：名量词(把/双/张/种/层/封/页/辆/节/所/班/段/头/句/公斤/斤/角/毛/刻/米/碗/盘) / 动量词(口/回/遍/声) / 量词重叠
- 连词：只有/只要/不但/而且/如果/可/可是/然后 / 或/或者
- 助词：的话

### 短语（12项）
- 固定短语：一般来说 / 不一会儿 / 不怎么样 / 在……看来 / 看上去 / 看来 / 看起来 / 越来越 / 不A不B
- 结构类型：动补短语2 / 同位短语 / 数量重叠

### 特殊表达法（2项）
- 概数表达法2(大概/相邻数词连用)

### 语素（2项）
- 前缀：老— / 后缀：—家/—子/—员
