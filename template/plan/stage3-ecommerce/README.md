## 语法对齐说明

本主题96个语法点与HSK3级完全一致，但所有例句和示范内容全部替换为电商业务场景。例如：
- HSK原句「你把箱子放在这儿吧」→ 电商「你把快递放在仓库的货架上吧」
- HSK原句「如果明天不下雨，我们就去爬山」→ 电商「如果订单量超过一百，我们就安排加班打包」
