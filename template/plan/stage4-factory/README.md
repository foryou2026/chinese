## 地域聚焦：东南亚制造业（进阶）

- **目标地区**：泰国（EEC东部经济走廊）、越南（北宁/平阳/龙安工业区）、印尼（勿加泗/芝卡朗/巴淡岛）、马来西亚（槟城/柔佛）、菲律宾（经济特区）

## 语法对齐说明

本主题95个语法点与HSK4级完全一致，但所有例句和示范内容全部替换为工厂业务场景。例如：
- HSK原句「既然你这么喜欢小孩儿，毕业以后就当老师吧」→ 工厂「既然你对精益生产这么感兴趣，入职以后就到改善组来吧」
- HSK原句「跟线下购物相比，上网买东西更符合年轻人的习惯」→ 工厂「跟传统手工作业相比，自动化产线的产能和良率都更高」
- HSK原句「无论在多么紧张的情况下，他都表现得非常冷静」→ 工厂「无论交期多么紧张，品质标准都不能降低」
