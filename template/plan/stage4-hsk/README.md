## HSK4 新增语法分类索引

### 句子成分（2项）
- 补语：可能补语2(动词+得/不+了) / 程度补语2(形容词/心理动词+死了/厉害)

### 句子类型（40项）
- 句类：反问句2(疑问代词构成 / 难道……吗)
- 复句-假设：……否则 / 要是……就 / 要是……否则
- 复句-因果：既然……就 / 由于……因此
- 复句-并列：一方面……另一方面 / 不是……而是 / 既……又/也
- 复句-承接：……于是 / 首先……其次 / 首先……然后
- 复句-条件：不管……都/也 / 无论……都/也
- 复句-目的：……好……
- 复句-紧缩：不……也…… / 无标记
- 复句-让步：即使……也 / 就是……也
- 复句-转折：X是X就是/不过 / ……不过 / 尽管……但是/可是 / ……然而
- 复句-选择：不是……就是
- 复句-递进：不仅/不光……还/而且 / ……并且 / ……甚至 / 连……也/都……更
- 特殊句型：兼语句2(表爱憎义/表称谓认定义/表致使) / 双重否定句
- 特殊句型："把"字句2(4种格式) / 比较句3(A不如B / 跟……相比) / 被动句2(叫/让)

### 固定格式（18项）
- X也得X不X也得X / X就X(点儿)吧 / X就是了 / X是X / X是X，Y是Y
- 一+量词+比+一+量词 / 为了……而…… / 你X你的吧 / 再也不/没……
- 动词+一X是一X / 在……方面 / 够……的 / 怎么都/也+不/没……
- 拿……来说 / (没)有什么(好)X的 / 让/叫你X你就X / 说什么/怎么着也得X / 还X呢

### 特殊表达法（3项）
- 小数/分数/百分数/倍数的表示法
- 概数表示法3(数词+来+量词)
- 用"大约、左右、前后"表示概数

### 短语（9项）
- 固定短语：不怎么 / 就是说/这就是说 / 有的是 / 来得及/来不及 / 说不定
- 四字格：一A一B / 无A无B / 有A有B
- 结构类型：数词+形容词+量词

### 词类（23项）
- 介词：引出凭借/依据(作为/按/按照) / 引出原因(由于1) / 引出对象(对于/与1) / 引出施事/受事(将/由2/叫/让) / 引出时间/处所(自/当/由1/随着/等1)
- 代词：指示代词(各/各位/各种/任何/此)
- 副词：关联(却) / 情态(也许/恐怕) / 方式(故意/互相/相互) / 时间(按时/从来/赶紧/赶快/已/忽然/将/将要/从/本来) / 程度(十分/更加/稍/稍微/尤其/多么) / 范围(共/全/光/仅/仅仅/至少) / 语气(才/就/并/还/竟然/究竟/正好/到底/难道/千万/确实/只好/差点儿) / 频率(重新/往往/偶尔/再次/老)
- 动词：能愿动词(敢)
- 助词：其他助词(等2/者/之/就是)
- 叹词：啊/嗯
- 连词：连接分句或句子(不光/不仅/另外/要是/因此/由于/加上/此外/而/既然/甚至/不过/并且) / 连接词或词组(并/而/与)
- 量词：动量词(场/顿/趟) / 名量词-专用(打/袋/棵/台/幅) / 名量词-借用(脸/手/盒/屋子/桌子)
