## 地域聚焦：东南亚制造业

- **目标地区**：泰国（罗勇工业区/东部经济走廊）、越南（北宁/平阳/胡志明工业区）、印尼（勿加泗/芝卡朗工业区）
- **工厂类型**：电子组装、服装鞋帽、食品加工、塑料注塑、五金机械、家具制造
- **语言场景**：中国管理层/技术人员与东南亚本地工人的中文沟通
- **文化要素**：东南亚劳动法规、宗教节假日（宋干节/开斋节/佛诞节）、热带气候生产管理、清真食品生产认证等

## 语法对齐说明

本主题96个语法点与HSK3级完全一致，但所有例句和示范内容全部替换为工厂业务场景。例如：
- HSK原句「你把箱子放在这儿吧」→ 工厂「你把零件放在工位上吧」
- HSK原句「如果明天不下雨，我们就去爬山」→ 工厂「如果原材料明天到，我们就安排加班生产」
- HSK原句「只有努力学习，才能得到好成绩」→ 工厂「只有通过质量检验，才能出货」
