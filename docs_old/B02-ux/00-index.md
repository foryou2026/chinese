## 0. 摘要

- **一句话定调**："给东南亚成年学习者的、像高级社交产品一样使用的汉语学习应用。"
- 风格 = **现代 / 克制 / 质感 / 趣味 / 沉浸**（5 个可验证形容词，任何设计决策必须命中 ≥ 3）。
- 视觉锚 = 黑白 + 红 + 全局毛玻璃；体验锚 = 不打断 / 不打扰 / 不模糊 / 不让等。
- 参照三家：小红书 + Stitch + Notion；反例三家：Duolingo + HelloChinese + 传统 ToB 后台。

## 2. 体验定调评分卡（用于评审任何 C04 原型 / C03 页面）

> 每张页面、每个组件、每个新交互上线前，用下表打分。任何一项 ≤ 2 直接打回。

| 维度 | 1 分 | 3 分 | 5 分 |
|------|------|------|------|
| 现代感 | 像 2015 教辅 | 像主流 SaaS | 像 Stitch / 小红书 2024+ |
| 克制度 | 颜色 / 控件 / emoji 满天飞 | 主从有别 | 黑白红 + 毛玻璃 + 1 个主操作 |
| 质感 | 平面 + 实色卡片 | 阴影 + 圆角 | 毛玻璃 + 高光内边 + 微噪 |
| 趣味 | 完全无激励 / 反馈 | 偶有微互动 | 有进度 / 连胜 / 微动效但**不打断**学习 |
| 沉浸 | 中心固定列宽 + 边缘苍白 | 自适应宽度 | 全屏宽度 + 远景光晕 + 真黑暗模式 |

总分 ≥ 18 / 25 即可上线。
