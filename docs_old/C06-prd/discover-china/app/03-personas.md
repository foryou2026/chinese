# 03 · 画像与场景 · discover-china / **app**

## 3.1 主画像

| 维度 | 设定 |
|------|------|
| 与 course 共用 | `learner` 角色;同设备 / 网络 / 5 语策略 |
| 阅读场景 | 通勤(5-10 min,听一两段)、咖啡时间(15-20 min,看一篇)、睡前(10 min) |
| 学习诉求 | 在课程之外补充文化背景;**不要求** 完整测验 |
| 付费 | 与 course 一致(订阅解锁全文化类目;免费可看每类目前 N 篇) |

## 3.2 场景矩阵

| 场景 | 路径 |
|------|------|
| 浏览新内容 | [P-001](06-page-specs/P-app-discover-china-001.md) → 选类目 |
| 找回上次阅读 | 首页"最近阅读" → [P-003](06-page-specs/P-app-discover-china-003.md) |
| 搜索文章标题 | [P-001](06-page-specs/P-app-discover-china-001.md) 顶部搜索栏(支持中文 + 拼音 + 5 语标题) |
| 逐句精读 | [P-003](06-page-specs/P-app-discover-china-003.md) → 点行播放 → 切语种对照 |

## 3.3 反角色

- 寻求系统化中文学习者 → 应去 course;
- 寻求中文教材内容者 → 本 feature 内容偏文化向,不替代教材。
